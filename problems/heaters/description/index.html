475. Heaters

Jump to Solution: Python Java C++ JavaScript C#

Problem Description

Winter is coming! During the contest, your first job is to design a standard heater with a fixed warm radius to warm all the houses.

Every house can be warmed, as long as the house is within the heater's warm radius range.

Given the positions of houses and heaters on a horizontal line, return the minimum radius standard of heaters so that those heaters could cover all houses.

Notice that all the heaters follow your radius standard, and the warm radius will be the same.

Example 1:

Input: houses = [1,2,3], heaters = [2]
Output: 1
Explanation: The only heater was placed in the position 2, and if we use the radius 1 standard, then all the houses can be warmed.

Example 2:

Input: houses = [1,2,3,4], heaters = [1,4]
Output: 1
Explanation: The two heaters were placed at positions 1 and 4. We need to use a radius 1 standard, then all the houses can be warmed.

Example 3:

Input: houses = [1,5], heaters = [2]
Output: 3

Constraints:

  • 1 <= houses.length, heaters.length <= 3 * 10⁴
  • 1 <= houses[i], heaters[i] <= 10⁹

Solution

Python Solution

class Solution:
    def findRadius(self, houses: List[int], heaters: List[int]) -> int:
        # Sort both arrays
        houses.sort()
        heaters.sort()
        
        # Initialize result and heater index
        result = 0
        i = 0
        
        # For each house, find the minimum distance to a heater
        for house in houses:
            # Find the closest heater
            while i < len(heaters) - 1 and abs(heaters[i] - house) >= abs(heaters[i + 1] - house):
                i += 1
            
            # Update result with maximum distance needed
            result = max(result, abs(heaters[i] - house))
        
        return result

Time Complexity: O(n log n + m log m)

Where n is the number of houses and m is the number of heaters. The sorting step dominates the time complexity.

Space Complexity: O(1)

Only constant extra space is used.

class Solution {

        Arrays.sort(houses);
        Arrays.sort(heaters);

        int i = 0;

                i++;
            }

        }
        
        return result;
    }
}
