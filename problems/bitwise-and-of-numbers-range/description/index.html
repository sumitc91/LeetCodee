201. Bitwise AND of Numbers Range

Medium

Problem Description

Given two integers left and right that represent the range [left, right], return the bitwise AND of all numbers in this range, inclusive.

Examples

Example 1:
Input: left = 5, right = 7
Output: 4
Explanation:
5 -> 101
6 -> 110
7 -> 111
The bitwise AND of all numbers is 4 -> 100

Example 2:
Input: left = 0, right = 0
Output: 0

Example 3:
Input: left = 1, right = 2147483647
Output: 0

Python Solution


def rangeBitwiseAnd(left: int, right: int) -> int:
    shift = 0
    # Find the common prefix
    while left != right:
        left >>= 1
        right >>= 1
        shift += 1
    return left << shift

Alternative Solution:


def rangeBitwiseAnd(left: int, right: int) -> int:
    # Brian Kernighan's Algorithm
    while right > left:
        right = right & (right - 1)
    return right

Complexity Analysis

  • Time Complexity: O(log n) where n is the maximum number of bits needed to represent the numbers
  • Space Complexity: O(1)

Solution Explanation

There are two main approaches to solve this problem:

  • Common Prefix Approach:
    • Find the common prefix of both numbers
    • Right shift until numbers are equal
    • Left shift by the same amount
    • More intuitive solution
  • Brian Kernighan's Algorithm:
    • Use property of n & (n-1)
    • Removes rightmost set bit
    • Continue until right ≤ left
    • More efficient for some cases

Key Insights

The key observation is that:

  • Only common prefix bits remain after AND
  • Any bit that differs becomes 0
  • All trailing bits become 0
  • Result is always ≤ smaller number

Common Pitfalls

  • Integer overflow
  • Negative numbers
  • Large ranges
  • Bit shift direction
