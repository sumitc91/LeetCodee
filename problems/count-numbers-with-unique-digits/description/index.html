357. Count Numbers with Unique Digits

Jump to Solution: Python Java C++ JavaScript C#

Problem Description

Given an integer n, return the count of all numbers with unique digits, x, where 0 <= x < 10ⁿ.

Example 1:

Input: n = 2
Output: 91
Explanation: The answer should be the total numbers in the range of 0 ≤ x < 100, excluding 11,22,33,44,55,66,77,88,99

Example 2:

Input: n = 0
Output: 1

Constraints:

  • 0 <= n <= 8

Solution

Python Solution

class Solution:
    def countNumbersWithUniqueDigits(self, n: int) -> int:
        if n == 0:
            return 1
        
        # For n = 1, count is 10 (0-9)
        result = 10
        
        # For n > 1, use permutation
        # First digit can be 1-9 (9 choices)
        # Second digit can be 0-9 except the first digit (9 choices)
        # Third digit can be 0-9 except used digits (8 choices)
        # And so on...
        available_digits = 9
        current_choices = 9
        
        while n > 1 and available_digits > 0:
            current_choices *= available_digits
            result += current_choices
            available_digits -= 1
            n -= 1
        
        return result

Time Complexity: O(n)

Where n is the input number.

Space Complexity: O(1)

Only uses a constant amount of extra space.

Approach Explanation

The solution uses combinatorics and permutation principles:

  1. Key Insight:
    • Base cases
    • Permutation rules
    • Digit choices
    • Accumulation
  2. Algorithm Steps:
    • Handle base case
    • Calculate choices
    • Apply permutation
    • Sum results

Mathematical Analysis:

  • n = 1: 10 numbers (0-9)
  • n = 2: 9 * 9 = 81 new numbers
  • n = 3: 9 * 9 * 8 = 648 new numbers
  • And so on...

Optimization insights:

  • Early stopping
  • No recursion
  • Constant space
  • Linear time

Edge Cases:

  • n = 0
  • n = 1
  • Maximum n
  • Leading zeros
