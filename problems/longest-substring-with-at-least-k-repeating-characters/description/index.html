395. Longest Substring with At Least K Repeating Characters

Jump to Solution: Python Java C++ JavaScript C#

Problem Description

Given a string s and an integer k, return the length of the longest substring of s such that the frequency of each character in this substring is greater than or equal to k.

If no such substring exists, return 0.

Example 1:

Input: s = "aaabb", k = 3
Output: 3
Explanation: The longest substring is "aaa", as 'a' is repeated 3 times.

Example 2:

Input: s = "ababbc", k = 2
Output: 5
Explanation: The longest substring is "ababb", as 'a' is repeated 2 times and 'b' is repeated 3 times.

Constraints:

  • 1 <= s.length <= 10⁴
  • s consists of only lowercase English letters.
  • 1 <= k <= 10⁵

Solution

Python Solution

class Solution:
    def longestSubstring(self, s: str, k: int) -> int:
        # Base cases
        if not s or k > len(s):
            return 0
        if k <= 1:
            return len(s)
        
        # Count character frequencies
        char_count = Counter(s)
        
        # Find first character with frequency < k
        for i, char in enumerate(s):
            if char_count[char] < k:
                # Split string at this character and solve recursively
                return max(self.longestSubstring(substr, k) 
                         for substr in s.split(char))
        
        # If we reach here, all characters have frequency >= k
        return len(s)

Time Complexity: O(n²)

Where n is the length of the string. In worst case, we split at each character.

Space Complexity: O(n)

For the recursive call stack and character counting.

Java Solution

class Solution {
    public int longestSubstring(String s, int k) {
        // Base cases
        if (s == null || s.length() == 0 || k > s.length()) {
            return 0;
        }
        if (k <= 1) {
            return s.length();
        }
        
        // Count character frequencies
        int[] counts = new int[26];
        for (char c : s.toCharArray()) {
            counts[c - 'a']++;
        }
        
        // Find first character with frequency < k
        boolean allValid = true;
        int splitIndex = 0;
        for (int i = 0; i < s.length() && allValid; i++) {
            if (counts[s.charAt(i) - 'a'] < k) {
                allValid = false;
                splitIndex = i;
            }
        }
        
        // If all characters have frequency >= k
        if (allValid) {
            return s.length();
        }
        
        // Split and solve recursively
        return Math.max(
            longestSubstring(s.substring(0, splitIndex), k),
            longestSubstring(s.substring(splitIndex + 1), k)
        );
    }
}
