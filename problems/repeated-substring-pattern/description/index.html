459. Repeated Substring Pattern

Jump to Solution: Python Java C++ JavaScript C#

Problem Description

Given a string s, check if it can be constructed by taking a substring of it and appending multiple copies of the substring together.

Example 1:

Input: s = "abab"
Output: true
Explanation: It is the substring "ab" twice.

Example 2:

Input: s = "aba"
Output: false

Example 3:

Input: s = "abcabcabcabc"
Output: true
Explanation: It is the substring "abc" four times or the substring "abcabc" twice.

Constraints:

  • 1 <= s.length <= 10⁴
  • s consists of lowercase English letters.

Solution

Python Solution

class Solution:
    def repeatedSubstringPattern(self, s: str) -> bool:
        # Concatenate string with itself and remove first and last characters
        doubled = s + s
        # Check if original string exists in the middle of doubled string
        return s in doubled[1:-1]

Time Complexity: O(n)

Where n is the length of the string.

Space Complexity: O(n)

For creating the doubled string.

Java Solution

class Solution {
    public boolean repeatedSubstringPattern(String s) {
        // Concatenate string with itself and remove first and last characters
        String doubled = s + s;
        // Check if original string exists in the middle of doubled string
        return doubled.substring(1, doubled.length() - 1).contains(s);
    }
}

Time Complexity: O(n)

Where n is the length of the string.

Space Complexity: O(n)

For creating the doubled string.

public class Solution {
    public bool RepeatedSubstringPattern(string s) {
        // Concatenate string with itself and remove first and last characters
        string doubled = s + s;
        // Check if original string exists in the middle of doubled string
        return doubled.Substring(1, doubled.Length - 2).Contains(s);

}
