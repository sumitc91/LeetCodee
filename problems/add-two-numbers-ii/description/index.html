445. Add Two Numbers II

Jump to Solution: Python Java C++ JavaScript C#

Problem Description

You are given two non-empty linked lists representing two non-negative integers. The most significant digit comes first and each of their nodes contains a single digit. Add the two numbers and return the sum as a linked list.

You may assume the two numbers do not contain any leading zero, except the number 0 itself.

Example 1:

Input: l1 = [7,2,4,3], l2 = [5,6,4]
Output: [7,8,0,7]
Explanation: 7243 + 564 = 7807.

Example 2:

Input: l1 = [2,4,3], l2 = [5,6,4]
Output: [8,0,7]
Explanation: 243 + 564 = 807.

Example 3:

Input: l1 = [0], l2 = [0]
Output: [0]

Constraints:

  • The number of nodes in each linked list is in the range [1, 100]
  • 0 <= Node.val <= 9
  • It is guaranteed that the list represents a number that does not have leading zeros.

Solution

Python Solution

class Solution:
    def addTwoNumbers(self, l1: Optional[ListNode], l2: Optional[ListNode]) -> Optional[ListNode]:
        # Convert linked lists to stacks
        s1, s2 = [], []
        while l1:
            s1.append(l1.val)
            l1 = l1.next
        while l2:
            s2.append(l2.val)
            l2 = l2.next
        
        # Add numbers from right to left
        carry = 0
        head = None
        
        while s1 or s2 or carry:
            # Get digits from stacks or use 0 if empty
            d1 = s1.pop() if s1 else 0
            d2 = s2.pop() if s2 else 0
            
            # Calculate sum and carry
            curr_sum = d1 + d2 + carry
            carry = curr_sum // 10
            digit = curr_sum % 10
            
            # Create new node and link it
            new_node = ListNode(digit)
            new_node.next = head
            head = new_node
        
        return head

Java Solution

class Solution {
    public ListNode addTwoNumbers(ListNode l1, ListNode l2) {
        // Convert linked lists to stacks
        Stack s1 = new Stack<>();
        Stack s2 = new Stack<>();
        
        while (l1 != null) {
            s1.push(l1.val);
            l1 = l1.next;
        }
        while (l2 != null) {
            s2.push(l2.val);
            l2 = l2.next;
        }
        
        // Add numbers from right to left
        int carry = 0;
        ListNode head = null;
        
        while (!s1.empty() || !s2.empty() || carry > 0) {
            // Get digits from stacks or use 0 if empty
            int d1 = !s1.empty() ? s1.pop() : 0;
            int d2 = !s2.empty() ? s2.pop() : 0;
            
            // Calculate sum and carry
            int currSum = d1 + d2 + carry;
            carry = currSum / 10;
            int digit = currSum % 10;
            
            // Create new node and link it
            ListNode newNode = new ListNode(digit);
            newNode.next = head;
            head = newNode;
        }
        
        return head;
    }
}

public class Solution {
    public ListNode AddTwoNumbers(ListNode l1, ListNode l2) {
        // Convert linked lists to stacks
        Stack s1 = new Stack();
        Stack s2 = new Stack();
        
        while (l1 != null) {
            s1.Push(l1.val);
            l1 = l1.next;
        }
        while (l2 != null) {
            s2.Push(l2.val);
            l2 = l2.next;
        }
        
        // Add numbers from right to left
        int carry = 0;
        ListNode head = null;
        
        while (s1.Count > 0 || s2.Count > 0 || carry > 0) {
            // Get digits from stacks or use 0 if empty
            int d1 = s1.Count > 0 ? s1.Pop() : 0;
            int d2 = s2.Count > 0 ? s2.Pop() : 0;
            
            // Calculate sum and carry
            int currSum = d1 + d2 + carry;
            carry = currSum / 10;
            int digit = currSum % 10;
            
            // Create new node and link it
            ListNode newNode = new ListNode(digit);
            newNode.next = head;
            head = newNode;
        }
        
        return head;
    }
}
