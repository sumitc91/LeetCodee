551. Student Attendance Record I

Jump to Solution: Python Java C++ JavaScript C#

Problem Description

You are given a string s containing only the characters 'A', 'L', and 'P' where:

  • 'A' stands for Absent
  • 'L' stands for Late
  • 'P' stands for Present

The student is eligible for an attendance award if they meet both of the following criteria:

  • The student was absent ('A') for strictly fewer than 2 days in total.
  • The student was never late ('L') for 3 or more consecutive days.

Return true if the student is eligible for an attendance award, or false otherwise.

Example 1:

Input: s = "PPALLP"
Output: true
Explanation: The student has fewer than 2 absences and was never late for 3 or more consecutive days.

Example 2:

Input: s = "PPALLL"
Output: false
Explanation: The student was late for 3 consecutive days in the last 3 days.

Constraints:

  • 1 <= s.length <= 1000
  • s[i] is either 'A', 'L', or 'P'

Solution

Python Solution

class Solution:
    def checkRecord(self, s: str) -> bool:
        absent_count = 0
        late_streak = 0
        
        for attendance in s:
            if attendance == 'A':
                absent_count += 1
                late_streak = 0
                if absent_count >= 2:
                    return False
            elif attendance == 'L':
                late_streak += 1
                if late_streak >= 3:
                    return False
            else:  # attendance == 'P'
                late_streak = 0
        
        return True

Time Complexity: O(n)

Where n is the length of the string. We scan through each character once.

Space Complexity: O(1)

We only use two variables regardless of input size.

        int absentCount = 0;
        int lateStreak = 0;

                lateStreak++;
                if (lateStreak >= 3) {
                    return false;
                }
            } else {  // attendance == 'P'
                lateStreak = 0;
            }
        }
        
        return true;
    }
}
