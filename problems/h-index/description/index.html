274. H-Index

Jump to Solution: Python Java C++ JavaScript C#

Problem Description

Given an array of integers citations where citations[i] is the number of citations a researcher received for their ith paper, return compute the researcher's h-index.

According to the definition of h-index on Wikipedia: A scientist has an index h if h of their n papers have at least h citations each, and the other n − h papers have no more than h citations each.

Example 1:

Input: citations = [3,0,6,1,5]
Output: 3
Explanation: [3,0,6,1,5] means the researcher has 5 papers in total and each of them had received 3, 0, 6, 1, 5 citations respectively.
Since the researcher has 3 papers with at least 3 citations each and the remaining two with no more than 3 citations each, their h-index is 3.

Example 2:

Input: citations = [1,3,1]
Output: 1

Constraints:

  • n == citations.length
  • 1 <= n <= 5000
  • 0 <= citations[i] <= 1000

Solution

Python Solution

class Solution:
    def hIndex(self, citations: List[int]) -> int:
        n = len(citations)
        count = [0] * (n + 1)
        
        # Count papers for each citation number
        for c in citations:
            if c >= n:
                count[n] += 1
            else:
                count[c] += 1
        
        # Find h-index by counting from right
        total = 0
        for i in range(n, -1, -1):
            total += count[i]
            if total >= i:
                return i
        return 0

Time Complexity: O(n)

We need to traverse the array twice.

Space Complexity: O(n)

We need an array of size n+1 to store the counts.

class Solution {
    public int hIndex(int[] citations) {
        int n = citations.length;
        int[] count = new int[n + 1];
        
        // Count papers for each citation number
        for (int c : citations) {

                count[c]++;
            }
        }
        
        // Find h-index by counting from right
        int total = 0;
        for (int i = n; i >= 0; i--) {
            total += count[i];
            if (total >= i) {
                return i;
            }
        }
        return 0;
    }
}

public class Solution {
    public int HIndex(int[] citations) {
        int n = citations.Length;
        int[] count = new int[n + 1];
        
        // Count papers for each citation number
        foreach (int c in citations) {
            if (c >= n) {
                count[n]++;
            } else {
                count[c]++;
            }
        }
        
        // Find h-index by counting from right
        int total = 0;
        for (int i = n; i >= 0; i--) {
            total += count[i];
            if (total >= i) {
                return i;
            }
        }
        return 0;
    }
}

Space Complexity: O(n)

Approach Explanation

The solution uses counting sort approach to find the h-index:

  1. Key Insight:
    • H-index cannot be larger than the number of papers
    • We can count papers for each citation number
    • Process counts from highest to lowest
  2. Algorithm Steps:
    • Create count array of size n+1
    • Count papers for each citation number
    • Process from right to find h-index

Example walkthrough for [3,0,6,1,5]:

  • Count array: [1,1,0,1,0,1,1] (index 5 represents 5 and above)
  • Process from right: 1+1+1=3 papers with 3+ citations
  • H-index is 3

Alternative Approaches:

  • Sorting approach (O(n log n))
  • Binary search approach
  • Direct counting approach

Edge Cases:

  • All zeros
  • All same numbers
  • Single element array
  • Citations greater than paper count
