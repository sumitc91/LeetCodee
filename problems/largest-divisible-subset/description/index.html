368. Largest Divisible Subset

Jump to Solution: Python Java C++ JavaScript C#

Problem Description

Given a set of distinct positive integers nums, return the largest subset answer such that every pair (answer[i], answer[j]) of elements in this subset satisfies:

  • answer[i] % answer[j] == 0, or
  • answer[j] % answer[i] == 0

If there are multiple solutions, return any of them.

Example 1:

Input: nums = [1,2,3]
Output: [1,2]
Explanation: [1,3] is also accepted.

Example 2:

Input: nums = [1,2,4,8]
Output: [1,2,4,8]

Constraints:

  • 1 <= nums.length <= 1000
  • 1 <= nums[i] <= 2 * 10⁹
  • All the integers in nums are unique.

Solution

Python Solution

class Solution:
    def largestDivisibleSubset(self, nums: List[int]) -> List[int]:
        if not nums:
            return []
        
        # Sort the array
        nums.sort()
        n = len(nums)
        
        # dp[i] stores the length of largest subset ending at nums[i]
        dp = [1] * n
        # prev[i] stores the previous index in the subset ending at nums[i]
        prev = [-1] * n
        
        # Find the maximum length and its ending index
        max_len = 1
        max_index = 0
        
        for i in range(1, n):
            for j in range(i):
                if nums[i] % nums[j] == 0 and dp[j] + 1 > dp[i]:
                    dp[i] = dp[j] + 1
                    prev[i] = j
                    if dp[i] > max_len:
                        max_len = dp[i]
                        max_index = i
        
        # Reconstruct the subset
        result = []
        while max_index != -1:
            result.append(nums[max_index])
            max_index = prev[max_index]
        
        return result[::-1]

Java Solution

class Solution {
    public List largestDivisibleSubset(int[] nums) {
        if (nums == null || nums.length == 0) {
            return new ArrayList<>();
        }
        
        // Sort the array
        Arrays.sort(nums);
        int n = nums.length;
        
        // dp[i] stores the length of largest subset ending at nums[i]
        int[] dp = new int[n];
        // prev[i] stores the previous index in the subset ending at nums[i]
        int[] prev = new int[n];
        Arrays.fill(dp, 1);
        Arrays.fill(prev, -1);
        
        // Find the maximum length and its ending index
        int maxLen = 1;
        int maxIndex = 0;
        
        for (int i = 1; i < n; i++) {
            for (int j = 0; j < i; j++) {
                if (nums[i] % nums[j] == 0 && dp[j] + 1 > dp[i]) {
                    dp[i] = dp[j] + 1;
                    prev[i] = j;
                    if (dp[i] > maxLen) {
                        maxLen = dp[i];
                        maxIndex = i;
                    }
                }
            }
        }
        
        // Reconstruct the subset
        List result = new ArrayList<>();
        while (maxIndex != -1) {
            result.add(nums[maxIndex]);
            maxIndex = prev[maxIndex];
        }
        
        Collections.reverse(result);
        return result;
    }
}

var largestDivisibleSubset = function(nums) {
    if (!nums.length) {
        return [];
    }
    
    // Sort the array
    nums.sort((a, b) => a - b);
    const n = nums.length;
    
    // dp[i] stores the length of largest subset ending at nums[i]
    const dp = new Array(n).fill(1);
    // prev[i] stores the previous index in the subset ending at nums[i]
    const prev = new Array(n).fill(-1);
    
    // Find the maximum length and its ending index
    let maxLen = 1;
    let maxIndex = 0;
    
    for (let i = 1; i < n; i++) {
        for (let j = 0; j < i; j++) {
            if (nums[i] % nums[j] === 0 && dp[j] + 1 > dp[i]) {
                dp[i] = dp[j] + 1;
                prev[i] = j;
                if (dp[i] > maxLen) {
                    maxLen = dp[i];
                    maxIndex = i;
                }
            }
        }
    }
    
    // Reconstruct the subset
    const result = [];
    while (maxIndex !== -1) {
        result.push(nums[maxIndex]);
        maxIndex = prev[maxIndex];
    }
    
    return result.reverse();
};
