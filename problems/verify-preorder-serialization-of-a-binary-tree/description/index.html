331. Verify Preorder Serialization of a Binary Tree

Jump to Solution: Python Java C++ JavaScript C#

Problem Description

One way to serialize a binary tree is to use preorder traversal. When we encounter a non-null node, we record the node's value. If it is a null node, we record using a sentinel value such as '#'.

For example, the above binary tree can be serialized to the string "9,3,4,#,#,1,#,#,2,#,6,#,#", where '#' represents a null node.

Given a string of comma-separated values preorder, return true if it is a correct preorder traversal serialization of a binary tree.

Example 1:

Input: preorder = "9,3,4,#,#,1,#,#,2,#,6,#,#"
Output: true

Example 2:

Input: preorder = "1,#"
Output: false

Example 3:

Input: preorder = "9,#,#,1"
Output: false

Constraints:

  • 1 <= preorder.length <= 10⁴
  • preorder consists of comma-separated values
  • Each value is either an integer or '#'
  • All integers in the preorder string are in the range [0, 100]

Solution

Python Solution

class Solution:
    def isValidSerialization(self, preorder: str) -> bool:
        # Split the string by comma
        nodes = preorder.split(',')
        
        # Initialize slots available
        slots = 1
        
        # Process each node
        for node in nodes:
            # Each node takes one slot
            slots -= 1
            
            # If no slots available, it's invalid
            if slots < 0:
                return False
            
            # Non-null node creates two new slots
            if node != '#':
                slots += 2
        
        # All slots should be used
        return slots == 0

Time Complexity: O(n)

Where n is the length of the preorder string.

Space Complexity: O(n)

For storing the split array of nodes.

C++ Solution

class Solution {
public:
    bool isValidSerialization(string preorder) {
        // Initialize slots available
        int slots = 1;
        int n = preorder.length();
        
        // Process each node
        for (int i = 0; i < n; i++) {
            if (preorder[i] == ',') continue;
            
            // Each node takes one slot
            slots--;
            
            // If no slots available, it's invalid
            if (slots < 0) {
                return false;
            }
            
            // Non-null node creates two new slots
            if (preorder[i] != '#') {
                slots += 2;
                // Skip the number
                while (i < n && preorder[i] != ',') i++;
            }
        }
        
        // All slots should be used
        return slots == 0;
    }
};
