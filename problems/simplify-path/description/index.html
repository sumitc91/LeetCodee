71. Simplify Path

Medium

Problem Description

Given a string path, which is an absolute path (starting with a slash '/') to a file or directory in a Unix-style file system, convert it to the simplified canonical path.

In a Unix-style file system, a period '.' refers to the current directory, a double period '..' refers to the directory up a level, and any multiple consecutive slashes (i.e. '//') are treated as a single slash '/'. For this problem, any other format of periods such as '...' are treated as file/directory names.

The canonical path should have the following format:

  • The path starts with a single slash '/'.
  • Any two directories are separated by a single slash '/'.
  • The path does not end with a trailing '/'.
  • The path only contains the directories on the path from the root directory to the target file or directory (i.e., no period '.' or double period '..')

Examples

Example 1:
Input: path = "/home/"
Output: "/home"
Explanation: Note that there is no trailing slash after the last directory name.

Example 2:
Input: path = "/../"
Output: "/"
Explanation: Going one level up from the root directory is a no-op, as the root level is the highest level you can go.

Example 3:
Input: path = "/home//foo/"
Output: "/home/foo"
Explanation: In the canonical path, multiple consecutive slashes are replaced by a single one.

Python Solution


def simplifyPath(path: str) -> str:
    # Split path by '/' and filter out empty strings
    components = [x for x in path.split('/') if x and x != '.']
    stack = []
    
    # Process each component
    for comp in components:
        if comp == '..':
            if stack:
                stack.pop()
        else:
            stack.append(comp)
    
    # Build canonical path
    return '/' + '/'.join(stack)

Java Solution


class Solution {
    public String simplifyPath(String path) {
        Stack stack = new Stack<>();
        String[] components = path.split("/");
        
        for (String comp : components) {
            if (comp.equals("") || comp.equals(".")) {
                continue;
            }
            if (comp.equals("..")) {
                if (!stack.isEmpty()) {
                    stack.pop();
                }
            } else {
                stack.push(comp);
            }
        }
        
        if (stack.isEmpty()) {
            return "/";
        }
        
        StringBuilder result = new StringBuilder();
        for (String dir : stack) {
            result.append("/").append(dir);
        }
        
        return result.toString();
    }
}

C++ Solution


class Solution {
public:
    string simplifyPath(string path) {
        vector stack;
        stringstream ss(path);
        string component;
        
        // Split path by '/' and process each component
        while (getline(ss, component, '/')) {
            if (component == "" || component == ".") {
                continue;
            }
            if (component == "..") {
                if (!stack.empty()) {
                    stack.pop_back();
                }
            } else {
                stack.push_back(component);
            }
        }
        
        // Build canonical path
        if (stack.empty()) {
            return "/";
        }
        
        string result;
        for (const string& dir : stack) {
            result += "/" + dir;
        }
        
        return result;
    }
};

JavaScript Solution


/**
 * @param {string} path
 * @return {string}
 */
var simplifyPath = function(path) {
    const stack = [];
    const components = path.split('/').filter(x => x && x !== '.');
    
    for (const comp of components) {
        if (comp === '..') {
            if (stack.length) {
                stack.pop();
            }
        } else {
            stack.push(comp);
        }
    }
    
    return '/' + stack.join('/');
};

C# Solution


public class Solution {
    public string SimplifyPath(string path) {
        Stack stack = new Stack();
        string[] components = path.Split('/', StringSplitOptions.RemoveEmptyEntries);
        
        foreach (string comp in components) {
            if (comp == ".") {
                continue;
            }
            if (comp == "..") {
                if (stack.Count > 0) {
                    stack.Pop();
                }
            } else {
                stack.Push(comp);
            }
        }
        
        if (stack.Count == 0) {
            return "/";
        }
        
        return "/" + string.Join("/", stack.Reverse());
    }
}

Complexity Analysis

  • Time Complexity: O(n) where n is the length of the path
  • Space Complexity: O(n) to store the components

Solution Explanation

This solution uses a stack to process path components:

  • Key steps:
    • Split path by '/'
    • Filter out empty and '.' components
    • Process '..' by popping stack
    • Build canonical path from stack
  • Special cases:
    • Empty path → "/"
    • Multiple slashes → single slash
    • '..' at root → no-op
    • Trailing slash → removed

Key points:

  • Stack-based approach
  • Handles all edge cases
  • Efficient string processing
  • Follows Unix path rules
