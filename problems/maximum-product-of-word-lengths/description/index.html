318. Maximum Product of Word Lengths

Jump to Solution: Python Java C++ JavaScript C#

Problem Description

Given a string array words, return the maximum value of length(word[i]) * length(word[j]) where the two words do not share common letters. If no such two words exist, return 0.

Example 1:

Input: words = ["abcw","baz","foo","bar","xtfn","abcdef"]
Output: 16
Explanation: The two words can be "abcw", "xtfn".

Example 2:

Input: words = ["a","ab","abc","d","cd","bcd","abcd"]
Output: 4
Explanation: The two words can be "ab", "cd".

Example 3:

Input: words = ["a","aa","aaa","aaaa"]
Output: 0
Explanation: No such pair of words.

Constraints:

  • 2 <= words.length <= 1000
  • 1 <= words[i].length <= 1000
  • words[i] consists only of lowercase English letters

Solution

Python Solution

class Solution:
    def maxProduct(self, words: List[str]) -> int:
        # Create bit masks for each word
        masks = {}
        for word in words:
            # Create bit mask for current word
            mask = 0
            for c in word:
                mask |= 1 << (ord(c) - ord('a'))
            # Store maximum length for this mask
            masks[mask] = max(masks.get(mask, 0), len(word))
        
        # Find maximum product
        max_product = 0
        for mask1 in masks:
            for mask2 in masks:
                # If no common letters (no bits in common)
                if mask1 & mask2 == 0:
                    max_product = max(max_product, 
                                   masks[mask1] * masks[mask2])
        
        return max_product

Time Complexity: O(n * k + m²)

Where n is the number of words, k is the average word length, and m is the number of unique masks.

Space Complexity: O(m)

Where m is the number of unique masks.

Java Solution

class Solution {
    public int maxProduct(String[] words) {
        int n = words.length;
        int[] masks = new int[n];
        int[] lengths = new int[n];
        
        // Create bit masks for each word
        for (int i = 0; i < n; i++) {
            String word = words[i];
            int mask = 0;
            for (char c : word.toCharArray()) {
                mask |= 1 << (c - 'a');
            }
            masks[i] = mask;
            lengths[i] = word.length();
        }
        
        // Find maximum product
        int maxProduct = 0;
        for (int i = 0; i < n; i++) {
            for (int j = i + 1; j < n; j++) {
                // If no common letters (no bits in common)
                if ((masks[i] & masks[j]) == 0) {
                    maxProduct = Math.max(maxProduct, 
                                       lengths[i] * lengths[j]);
                }
            }
        }
        
        return maxProduct;
    }
}
