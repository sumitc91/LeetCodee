451. Sort Characters By Frequency

Jump to Solution: Python Java C++ JavaScript C#

Problem Description

Given a string s, sort it in decreasing order based on the frequency of the characters. The frequency of a character is the number of times it appears in the string.

Return the sorted string. If there are multiple answers, return any of them.

Example 1:

Input: s = "tree"
Output: "eert"
Explanation: 'e' appears twice while 'r' and 't' both appear once.
So 'e' should appear before both 'r' and 't'. Therefore "eetr" is also a valid answer.

Example 2:

Input: s = "cccaaa"
Output: "cccaaa"
Explanation: Both 'c' and 'a' appear three times, so both "cccaaa" and "aaaccc" are valid answers.
Note that "cacaca" is incorrect, as the same characters must be together.

Example 3:

Input: s = "Aabb"
Output: "bbAa"
Explanation: "bbaA" is also a valid answer, but "Aabb" is incorrect.
Note that 'A' and 'a' are treated as two different characters.

Constraints:

  • 1 <= s.length <= 5 * 10⁵
  • s consists of uppercase and lowercase English letters and digits.

Solution

Python Solution

class Solution:
    def frequencySort(self, s: str) -> str:
        # Count character frequencies
        freq = Counter(s)
        
        # Sort characters by frequency in descending order
        sorted_chars = sorted(freq.items(), key=lambda x: (-x[1], x[0]))
        
        # Build the result string
        result = []
        for char, count in sorted_chars:
            result.append(char * count)
        
        return ''.join(result)

Time Complexity: O(n log k)

Where n is the length of the string and k is the number of unique characters.

Space Complexity: O(k)

For storing the frequency map and sorted characters.

Java Solution

class Solution {
    public String frequencySort(String s) {
        // Count character frequencies
        Map freq = new HashMap<>();
        for (char c : s.toCharArray()) {
            freq.put(c, freq.getOrDefault(c, 0) + 1);
        }
        
        // Create a list of characters and sort by frequency
        List chars = new ArrayList<>(freq.keySet());
        Collections.sort(chars, (a, b) -> {
            int freqCompare = freq.get(b).compareTo(freq.get(a));
            return freqCompare != 0 ? freqCompare : a.compareTo(b);
        });
        
        // Build the result string
        StringBuilder result = new StringBuilder();
        for (char c : chars) {
            result.append(String.valueOf(c).repeat(freq.get(c)));
        }
        
        return result.toString();
    }
}

class Solution {
public:
    string frequencySort(string s) {
        // Count character frequencies
        unordered_map freq;
        for (char c : s) {
            freq[c]++;
        }
        
        // Sort characters by frequency
        vector> chars;
        for (const auto& p : freq) {
            chars.push_back(p);
        }
        sort(chars.begin(), chars.end(), 
             [](const pair& a, const pair& b) {
                 return a.second > b.second || (a.second == b.second && a.first < b.first);
             });
        
        // Build the result string
        string result;
        for (const auto& p : chars) {
            result.append(p.second, p.first);
        }
        
        return result;

/**
 * @param {string} s
 * @return {string}
 */
var frequencySort = function(s) {
    // Count character frequencies
    const freq = new Map();
    for (const c of s) {
        freq.set(c, (freq.get(c) || 0) + 1);
    }
    
    // Sort characters by frequency
    const chars = Array.from(freq.entries());
    chars.sort((a, b) => b[1] - a[1] || a[0].localeCompare(b[0]));
    
    // Build the result string
    return chars.map(([char, count]) => char.repeat(count)).join('');
};

C# Solution

public class Solution {
    public string FrequencySort(string s) {
        // Count character frequencies
        Dictionary freq = new Dictionary();
        foreach (char c in s) {
            if (!freq.ContainsKey(c)) {
                freq[c] = 0;
            }
            freq[c]++;
        }
        
        // Sort characters by frequency
        var chars = freq.ToList();
        chars.Sort((a, b) => {
            int freqCompare = b.Value.CompareTo(a.Value);
            return freqCompare != 0 ? freqCompare : a.Key.CompareTo(b.Key);
        });
        
        // Build the result string
        StringBuilder result = new StringBuilder();
        foreach (var pair in chars) {
            result.Append(new string(pair.Key, pair.Value));
        }
        
        return result.ToString();
    }
}
