523. Continuous Subarray Sum

Jump to Solution: Python Java C++ JavaScript C#

Problem Description

Given an integer array nums and an integer k, return true if nums has a continuous subarray of size at least two whose elements sum up to a multiple of k, or false otherwise.

An integer x is a multiple of k if there exists an integer n such that x = n * k. 0 is always a multiple of k.

Example 1:

Input: nums = [23,2,4,6,7], k = 6
Output: true
Explanation: [2, 4] is a continuous subarray of size 2 whose elements sum up to 6.

Example 2:

Input: nums = [23,2,6,4,7], k = 6
Output: true
Explanation: [23, 2, 6, 4, 7] is an continuous subarray of size 5 whose elements sum up to 42.
42 is a multiple of 6 because 42 = 7 * 6 and 7 is an integer.

Example 3:

Input: nums = [23,2,6,4,7], k = 13
Output: false

Constraints:

  • 1 <= nums.length <= 10⁵
  • 0 <= nums[i] <= 10⁹
  • 0 <= sum(nums[i]) <= 2³¹ - 1
  • 1 <= k <= 2³¹ - 1

Solution

Python Solution

class Solution:
    def checkSubarraySum(self, nums: List[int], k: int) -> bool:
        # Map to store remainder and its index
        remainder_map = {0: -1}
        curr_sum = 0
        
        for i, num in enumerate(nums):
            curr_sum += num
            
            # Get remainder when divided by k
            if k != 0:
                curr_sum = curr_sum % k
            
            # If remainder seen before and subarray size >= 2
            if curr_sum in remainder_map:
                if i - remainder_map[curr_sum] >= 2:
                    return True
            else:
                remainder_map[curr_sum] = i
        
        return False

Time Complexity: O(n)

Where n is the length of the input array. We traverse the array once.

Space Complexity: O(min(n,k))

For storing remainders in the hash map. The number of possible remainders is bounded by k.

Java Solution

class Solution {
    public boolean checkSubarraySum(int[] nums, int k) {
        // Map to store remainder and its index
        Map remainderMap = new HashMap<>();
        remainderMap.put(0, -1);
        int currSum = 0;
        
        for (int i = 0; i < nums.length; i++) {
            currSum += nums[i];
            
            // Get remainder when divided by k
            if (k != 0) {
                currSum = currSum % k;
            }
            
            // If remainder seen before and subarray size >= 2
            if (remainderMap.containsKey(currSum)) {
                if (i - remainderMap.get(currSum) >= 2) {
                    return true;
                }
            } else {
                remainderMap.put(currSum, i);
            }
        }
        
        return false;
    }
}
