481. Magical String

Jump to Solution: Python Java C++ JavaScript C#

Problem Description

A magical string s consists of only '1' and '2' and obeys the following rules:

  • The string s is magical because concatenating the number of contiguous occurrences of characters '1' and '2' generates the string s itself.

The first few elements of string s is s = "1221121221221121122……"

If we group the consecutive '1's and '2's in s, it will be "1 22 11 2 1 22 1 22 11 2 11 22 ......" and the occurrences of '1's or '2's in each group are "1 2 2 1 1 2 1 2 2 1 2 2 ......"

Given an integer n, return the number of '1's in the first n number in the magical string s.

Example 1:

Input: n = 6
Output: 3
Explanation: The first 6 elements of magical string s is "122112" and it contains three 1's, so return 3.

Example 2:

Input: n = 1
Output: 1

Constraints:

  • 1 <= n <= 10⁵

Solution

Python Solution

class Solution:
    def magicalString(self, n: int) -> int:
        if n <= 0:
            return 0
        if n <= 3:
            return 1
            
        # Initialize the magical string with first 3 elements
        s = [1, 2, 2]
        count = 1  # Count of 1's
        i = 2  # Index for generating next numbers
        
        # Generate magical string until length n
        while len(s) < n:
            # Current number to append (alternate between 1 and 2)
            num = 3 - s[-1]  # If last is 1, append 2; if last is 2, append 1
            
            # Append the number s[i] times
            for _ in range(s[i]):
                s.append(num)
                if num == 1 and len(s) <= n:
                    count += 1
            
            i += 1
        
        return count

Time Complexity: O(n)

We need to generate the string up to length n.

Space Complexity: O(n)

We store the generated string of length n.

Java Solution

class Solution {
    public int magicalString(int n) {
        if (n <= 0) {
            return 0;
        }
        if (n <= 3) {
            return 1;
        }
        
        // Initialize the magical string with first 3 elements
        int[] s = new int[n];
        s[0] = 1;
        s[1] = 2;
        s[2] = 2;
        
        int count = 1;  // Count of 1's
        int i = 2;      // Index for generating next numbers
        int end = 3;    // Current end of the array
        
        // Generate magical string until length n
        while (end < n) {
            // Current number to append (alternate between 1 and 2)
            int num = 3 - s[end - 1];  // If last is 1, append 2; if last is 2, append 1
            
            // Append the number s[i] times
            for (int j = 0; j < s[i] && end < n; j++) {
                s[end] = num;
                if (num == 1) {
                    count++;
                }
                end++;
            }
            
            i++;
        }
        
        return count;
    }
}

var magicalString = function(n) {
    if (n <= 0) {
        return 0;
    }
    if (n <= 3) {
        return 1;
    }
    
    // Initialize the magical string with first 3 elements
    const s = [1, 2, 2];
    let count = 1;  // Count of 1's

    while (s.length < n) {
        // Current number to append (alternate between 1 and 2)

        // Append the number s[i] times
        for (let j = 0; j < s[i] && s.length < n; j++) {
            s.push(num);
            if (num === 1) {
                count++;
            }
        }
        
        i++;
    }
    
    return count;
};
