539. Minimum Time Difference

Jump to Solution: Python Java C++ JavaScript C#

Problem Description

Given a list of 24-hour clock time points in "HH:MM" format, return the minimum minutes difference between any two time points in the list.

Example 1:

Input: timePoints = ["23:59","00:00"]
Output: 1
Explanation: The minimum difference is 1 minute between 23:59 and 00:00.

Example 2:

Input: timePoints = ["00:00","23:59","00:00"]
Output: 0
Explanation: The minimum difference is 0 minutes between 00:00 and 00:00.

Constraints:

  • 2 <= timePoints.length <= 2 * 10⁴
  • timePoints[i] is in the format "HH:MM"

Solution

Python Solution

class Solution:
    def findMinDifference(self, timePoints: List[str]) -> int:
        # Convert all times to minutes since midnight
        minutes = []
        for time in timePoints:
            h, m = map(int, time.split(':'))
            minutes.append(h * 60 + m)
        
        # Sort the minutes
        minutes.sort()
        
        # Find minimum difference
        min_diff = float('inf')
        for i in range(1, len(minutes)):
            min_diff = min(min_diff, minutes[i] - minutes[i-1])
        
        # Check difference between first and last time (wrapping around)
        min_diff = min(min_diff, 1440 + minutes[0] - minutes[-1])
        
        return min_diff

Where n is the number of time points. The sorting step dominates the time complexity.

For storing the minutes array.

class Solution {
    public int findMinDifference(List timePoints) {
        // Convert all times to minutes since midnight
        List minutes = new ArrayList<>();
        for (String time : timePoints) {
            String[] parts = time.split(":");
            int h = Integer.parseInt(parts[0]);
            int m = Integer.parseInt(parts[1]);
            minutes.add(h * 60 + m);
        }
        
        // Sort the minutes
        Collections.sort(minutes);
        
        // Find minimum difference
        int minDiff = Integer.MAX_VALUE;
        for (int i = 1; i < minutes.size(); i++) {
            minDiff = Math.min(minDiff, minutes.get(i) - minutes.get(i-1));
        }
        
        // Check difference between first and last time (wrapping around)
        minDiff = Math.min(minDiff, 1440 + minutes.get(0) - minutes.get(minutes.size()-1));
        
        return minDiff;
    }
}

        // Check difference between first and last time (wrapping around)
        minDiff = Math.Min(minDiff, 1440 + minutes[0] - minutes[minutes.Count-1]);
        
        return minDiff;
