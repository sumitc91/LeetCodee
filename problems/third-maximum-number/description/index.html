414. Third Maximum Number

Jump to Solution: Python Java C++ JavaScript C#

Problem Description

Given an integer array nums, return the third distinct maximum number in this array. If the third maximum does not exist, return the maximum number.

Example 1:

Input: nums = [3,2,1]
Output: 1
Explanation:
The first distinct maximum is 3.
The second distinct maximum is 2.
The third distinct maximum is 1.

Example 2:

Input: nums = [1,2]
Output: 2
Explanation:
The first distinct maximum is 2.
The second distinct maximum is 1.
The third distinct maximum does not exist, so the maximum (2) is returned instead.

Example 3:

Input: nums = [2,2,3,1]
Output: 1
Explanation:
The first distinct maximum is 3.
The second distinct maximum is 2 (both 2's are counted together since they have the same value).
The third distinct maximum is 1.

Constraints:

  • 1 <= nums.length <= 10⁴
  • -2³¹ <= nums[i] <= 2³¹ - 1

Solution

Python Solution

class Solution:
    def thirdMax(self, nums: List[int]) -> int:
        # Use a set to store unique numbers
        nums = set(nums)
        
        # If less than 3 unique numbers, return maximum
        if len(nums) < 3:
            return max(nums)
        
        # Remove maximum twice to get third maximum
        nums.remove(max(nums))
        nums.remove(max(nums))
        return max(nums)

Time Complexity: O(n)

Where n is the length of nums. Converting to set and finding maximum are O(n) operations.

Space Complexity: O(n)

For storing unique numbers in the set.

Java Solution

class Solution {
    public int thirdMax(int[] nums) {
        // Use TreeSet to maintain sorted unique numbers
        TreeSet set = new TreeSet<>();
        
        // Add all numbers to set
        for (int num : nums) {
            set.add(num);
            // Keep only top 3 numbers
            if (set.size() > 3) {
                set.remove(set.first());
            }
        }
        
        // Return third maximum if exists, otherwise maximum
        return set.size() < 3 ? set.last() : set.first();
    }
}

Time Complexity: O(n log k)

Where n is the length of nums and k is 3. Each insertion into TreeSet takes O(log k) time.

Space Complexity: O(k)

Where k is 3, for storing the top 3 numbers.

/**

var thirdMax = function(nums) {
    // Use Set to store unique numbers
    const set = new Set(nums);
    
    // If less than 3 unique numbers, return maximum
    if (set.size < 3) {
        return Math.max(...set);
    }
    
    // Remove maximum twice to get third maximum
    set.delete(Math.max(...set));
    set.delete(Math.max(...set));
    return Math.max(...set);
};

Where n is the length of nums. Converting to Set and finding maximum are O(n) operations.

Space Complexity: O(n)

For storing unique numbers in the Set.

C# Solution

public class Solution {
    public int ThirdMax(int[] nums) {
        // Use SortedSet to maintain sorted unique numbers
        SortedSet set = new SortedSet();
        
        // Add all numbers to set
        foreach (int num in nums) {
            set.Add(num);
            // Keep only top 3 numbers
            if (set.Count > 3) {
                set.Remove(set.Min);
            }
        }
        
        // Return third maximum if exists, otherwise maximum
        return set.Count < 3 ? set.Max : set.Min;
    }
}

Time Complexity: O(n log k)

Where n is the length of nums and k is 3. Each insertion into SortedSet takes O(log k) time.

Space Complexity: O(k)

Where k is 3, for storing the top 3 numbers.
