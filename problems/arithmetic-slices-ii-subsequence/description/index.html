446. Arithmetic Slices II - Subsequence

Jump to Solution: Python Java C++ JavaScript C#

Problem Description

Given an integer array nums, return the number of all the arithmetic subsequences of nums.

A sequence of numbers is called arithmetic if it consists of at least three elements and if the difference between any two consecutive elements is the same.

  • For example, [1, 3, 5, 7] is arithmetic because the differences between consecutive elements are all 2.
  • However, [1, 1, 2, 5, 7] is not arithmetic.

A subsequence of an array is a sequence that can be formed by removing some elements (possibly none) of the array.

Example 1:

Input: nums = [2,4,6,8,10]
Output: 7
Explanation: All arithmetic subsequence slices are:
[2,4,6]
[4,6,8]
[6,8,10]
[2,4,6,8]
[4,6,8,10]
[2,4,6,8,10]
[2,6,10]

Example 2:

Input: nums = [7,7,7,7,7]
Output: 16
Explanation: Any subsequence of this array is arithmetic.

Constraints:

  • 1 <= nums.length <= 1000
  • -2³¹ <= nums[i] <= 2³¹ - 1

Solution

Python Solution

class Solution:
    def numberOfArithmeticSlices(self, nums: List[int]) -> int:
        n = len(nums)
        total = 0
        dp = [defaultdict(int) for _ in range(n)]
        
        for i in range(1, n):
            for j in range(i):
                diff = nums[i] - nums[j]
                # Add sequences ending at j with the same difference
                total += dp[j][diff]
                # Add new sequence ending at i
                dp[i][diff] += dp[j][diff] + 1
        
        return total

Time Complexity: O(n²)

Where n is the length of the input array.

Space Complexity: O(n²)

For storing the dynamic programming states.

Java Solution

class Solution {
    public int numberOfArithmeticSlices(int[] nums) {
        int n = nums.length;
        int total = 0;
        Map[] dp = new HashMap[n];
        
        for (int i = 0; i < n; i++) {
            dp[i] = new HashMap<>();
        }
        
        for (int i = 1; i < n; i++) {
            for (int j = 0; j < i; j++) {
                long diff = (long)nums[i] - nums[j];
                if (diff <= Integer.MIN_VALUE || diff > Integer.MAX_VALUE)
                    continue;
                    
                int d = (int)diff;
                int sum = dp[j].getOrDefault(d, 0);
                int origin = dp[i].getOrDefault(d, 0);
                
                dp[i].put(d, origin + sum + 1);
                total += sum;
            }
        }
        
        return total;
    }
}

C# Solution

public class Solution {
    public int NumberOfArithmeticSlices(int[] nums) {
        int n = nums.Length;
        int total = 0;
        Dictionary[] dp = new Dictionary[n];
        
        for (int i = 0; i < n; i++) {
            dp[i] = new Dictionary();
        }
        
        for (int i = 1; i < n; i++) {
            for (int j = 0; j < i; j++) {
                long diff = (long)nums[i] - nums[j];
                if (diff <= int.MinValue || diff > int.MaxValue)
                    continue;
                    
                int d = (int)diff;
                int sum = dp[j].ContainsKey(d) ? dp[j][d] : 0;
                int origin = dp[i].ContainsKey(d) ? dp[i][d] : 0;
                
                dp[i][d] = origin + sum + 1;
                total += sum;
            }
        }
        
        return total;
    }
}
