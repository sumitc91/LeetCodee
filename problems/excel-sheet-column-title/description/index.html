168. Excel Sheet Column Title

Easy

Problem Description

Given an integer columnNumber, return its corresponding column title as it appears in an Excel sheet.

For example:

A -> 1
B -> 2
C -> 3
...
Z -> 26
AA -> 27
AB -> 28 
...

Examples

Example 1:
Input: columnNumber = 1
Output: "A"

Example 2:
Input: columnNumber = 28
Output: "AB"

Example 3:
Input: columnNumber = 701
Output: "ZY"
Jump to Solution: Python Java C++ JavaScript C#

Python Solution


def convertToTitle(columnNumber: int) -> str:
    result = []
    
    while columnNumber > 0:
        columnNumber -= 1
        result.append(chr(columnNumber % 26 + ord('A')))
        columnNumber //= 26
    
    return ''.join(reversed(result))

Java Solution


class Solution {
    public String convertToTitle(int columnNumber) {
        StringBuilder result = new StringBuilder();
        
        while (columnNumber > 0) {
            columnNumber--;
            result.append((char)('A' + columnNumber % 26));
            columnNumber /= 26;
        }
        
        return result.reverse().toString();
    }
}

JavaScript Solution


/**
 * @param {number} columnNumber
 * @return {string}
 */
var convertToTitle = function(columnNumber) {
    const result = [];
    
    while (columnNumber > 0) {
        columnNumber--;
        result.push(String.fromCharCode('A'.charCodeAt(0) + columnNumber % 26));
        columnNumber = Math.floor(columnNumber / 26);
    }
    
    return result.reverse().join('');
};

C# Solution


public class Solution {
    public string ConvertToTitle(int columnNumber) {
        StringBuilder result = new StringBuilder();
        
        while (columnNumber > 0) {
            columnNumber--;
            result.Append((char)('A' + columnNumber % 26));
            columnNumber /= 26;
        }
        
        return new string(result.ToString().Reverse().ToArray());
    }
}

Complexity Analysis

  • Time Complexity: O(log n) where n is the input number
  • Space Complexity: O(log n) to store the result string

Solution Explanation

This solution converts from base-10 to base-26:

  • Key concept:
    • Base conversion
    • Character mapping
    • Zero-based indexing
  • Algorithm steps:
    • Adjust for zero-based
    • Convert remainder
    • Build string
    • Reverse result

Key points:

  • Logarithmic time
  • Handle zero-based
  • String building
  • Reverse order
