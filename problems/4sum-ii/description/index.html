454. 4Sum II

Jump to Solution: Python Java C++ JavaScript C#

Problem Description

Given four integer arrays nums1, nums2, nums3, and nums4 all of length n, return the number of tuples (i, j, k, l) such that:

  • 0 <= i, j, k, l < n
  • nums1[i] + nums2[j] + nums3[k] + nums4[l] == 0

Example 1:

Input: nums1 = [1,2], nums2 = [-2,-1], nums3 = [-1,2], nums4 = [0,2]
Output: 2
Explanation:
The two tuples are:
1. (0, 0, 0, 1) -> nums1[0] + nums2[0] + nums3[0] + nums4[1] = 1 + (-2) + (-1) + 2 = 0
2. (1, 1, 0, 0) -> nums1[1] + nums2[1] + nums3[0] + nums4[0] = 2 + (-1) + (-1) + 0 = 0

Example 2:

Input: nums1 = [0], nums2 = [0], nums3 = [0], nums4 = [0]
Output: 1

Constraints:

  • n == nums1.length
  • n == nums2.length
  • n == nums3.length
  • n == nums4.length
  • 1 <= n <= 200
  • -2²⁸ <= nums1[i], nums2[i], nums3[i], nums4[i] <= 2²⁸

Solution

Python Solution

class Solution:
    def fourSumCount(self, nums1: List[int], nums2: List[int], nums3: List[int], nums4: List[int]) -> int:
        # Create a hash map for sums of first two arrays
        sum_map = {}
        for a in nums1:
            for b in nums2:
                sum_map[a + b] = sum_map.get(a + b, 0) + 1
        
        # Count tuples by checking complement sums
        count = 0
        for c in nums3:
            for d in nums4:
                count += sum_map.get(-(c + d), 0)
        
        return count

Time Complexity: O(n²)

Where n is the length of the input arrays.

Space Complexity: O(n²)

For storing sums in the hash map.

Java Solution

class Solution {
    public int fourSumCount(int[] nums1, int[] nums2, int[] nums3, int[] nums4) {
        // Create a hash map for sums of first two arrays
        Map sumMap = new HashMap<>();
        for (int a : nums1) {
            for (int b : nums2) {
                sumMap.merge(a + b, 1, Integer::sum);
            }
        }
        
        // Count tuples by checking complement sums
        int count = 0;
        for (int c : nums3) {
            for (int d : nums4) {
                count += sumMap.getOrDefault(-(c + d), 0);
            }
        }
        
        return count;
    }
}

var fourSumCount = function(nums1, nums2, nums3, nums4) {
    // Create a hash map for sums of first two arrays
    const sumMap = new Map();
    for (const a of nums1) {
        for (const b of nums2) {
            const sum = a + b;
            sumMap.set(sum, (sumMap.get(sum) || 0) + 1);
        }
    }
    
    // Count tuples by checking complement sums
    let count = 0;
    for (const c of nums3) {
        for (const d of nums4) {
            count += sumMap.get(-(c + d)) || 0;
        }
    }
    
    return count;
};

C# Solution

public class Solution {
    public int FourSumCount(int[] nums1, int[] nums2, int[] nums3, int[] nums4) {
        // Create a hash map for sums of first two arrays
        Dictionary sumMap = new Dictionary();
        foreach (int a in nums1) {
            foreach (int b in nums2) {
                int sum = a + b;
                if (!sumMap.ContainsKey(sum)) {
                    sumMap[sum] = 0;
                }
                sumMap[sum]++;
            }
        }
        
        // Count tuples by checking complement sums
        int count = 0;
        foreach (int c in nums3) {
            foreach (int d in nums4) {
                int complement = -(c + d);
                if (sumMap.ContainsKey(complement)) {
                    count += sumMap[complement];
                }
            }
        }
        
        return count;
    }
}

Where n is the length of the input arrays.

Space Complexity: O(n²)

For storing sums in the hash map.
