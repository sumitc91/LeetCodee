537. Complex Number Multiplication

Jump to Solution: Python Java C++ JavaScript C#

Problem Description

A complex number can be represented as a string on the form "real+imaginaryi" where:

  • real is the real part and is an integer in the range [-100, 100].
  • imaginary is the imaginary part and is an integer in the range [-100, 100].
  • i² equals -1.

Given two complex numbers num1 and num2 as strings, return a string of the complex number that represents their multiplications.

Example 1:

Input: num1 = "1+1i", num2 = "1+1i"
Output: "0+2i"
Explanation: (1 + i) * (1 + i) = 1 + i² + 2 * i = 2i

Example 2:

Input: num1 = "1+-1i", num2 = "1+-1i"
Output: "0+-2i"
Explanation: (1 - i) * (1 - i) = 1 + i² - 2 * i = -2i

Constraints:

  • num1 and num2 are valid complex numbers
  • The real and imaginary parts of num1 and num2 are in the range [-100, 100]

Solution

Python Solution

class Solution:
    def complexNumberMultiply(self, num1: str, num2: str) -> str:
        # Parse complex numbers
        a1, b1 = map(int, num1[:-1].split('+'))
        a2, b2 = map(int, num2[:-1].split('+'))
        
        # Multiply complex numbers: (a1 + b1i)(a2 + b2i)
        # = (a1a2 - b1b2) + (a1b2 + a2b1)i
        real = a1 * a2 - b1 * b2
        imag = a1 * b2 + a2 * b1
        
        return f"{real}+{imag}i"

Time Complexity: O(1)

The operations are constant time as we're just parsing and performing arithmetic.

Space Complexity: O(1)

We only use a constant amount of extra space.

Java Solution

class Solution {
    public String complexNumberMultiply(String num1, String num2) {
        // Parse complex numbers
        String[] parts1 = num1.substring(0, num1.length() - 1).split("\\+");
        String[] parts2 = num2.substring(0, num2.length() - 1).split("\\+");
        
        int a1 = Integer.parseInt(parts1[0]);
        int b1 = Integer.parseInt(parts1[1]);
        int a2 = Integer.parseInt(parts2[0]);
        int b2 = Integer.parseInt(parts2[1]);
        
        // Multiply complex numbers
        int real = a1 * a2 - b1 * b2;
        int imag = a1 * b2 + a2 * b1;
        
        return real + "+" + imag + "i";
    }
}
