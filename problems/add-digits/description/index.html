258. Add Digits

Jump to Solution: Python Java C++ JavaScript C#

Problem Description

Given an integer num, repeatedly add all its digits until the result has only one digit, and return it.

Example 1:

Input: num = 38
Output: 2
Explanation: The process is
38 --> 3 + 8 --> 11
11 --> 1 + 1 --> 2 
Since 2 has only one digit, return it.

Example 2:

Input: num = 0
Output: 0

Constraints:

  • 0 <= num <= 2³¹ - 1

Follow up:

Could you do it without any loop/recursion in O(1) runtime?

Solution

Python Solution

class Solution:
    def addDigits(self, num: int) -> int:
        # Digital Root Formula: dr(n) = 1 + ((n-1) mod 9)
        if num == 0:
            return 0
        return 1 + ((num - 1) % 9)

Time Complexity: O(1)

Using the digital root formula, we can solve it in constant time.

Space Complexity: O(1)

Only uses a constant amount of space.

Approach Explanation

The solution uses the concept of digital root in mathematics:

  1. Key Insight:
    • Digital root has a mathematical pattern
    • For any number n, dr(n) = 1 + ((n-1) mod 9)
    • Special case: when n = 0, dr(n) = 0
  2. Mathematical Proof:
    • Any number can be written as: n = d₀ + d₁×10 + d₂×100 + ...
    • 10 ≡ 1 (mod 9), 100 ≡ 1 (mod 9), etc.
    • Therefore, n ≡ d₀ + d₁ + d₂ + ... (mod 9)
    • This leads to the formula: dr(n) = 1 + ((n-1) mod 9)

Example walkthrough for num = 38:

  • 38 - 1 = 37
  • 37 % 9 = 1
  • 1 + 1 = 2
  • Verify: 38 → 3+8 = 11 → 1+1 = 2

Key insights:

  • O(1) solution without loops
  • Based on number theory
  • Handles all cases correctly
  • Special case for zero

Alternative Iterative Solution: 

while (num > 9) {

    }
    num = sum;
