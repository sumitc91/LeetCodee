306. Additive Number

Jump to Solution: Python Java C++ JavaScript C#

Problem Description

An additive number is a string whose digits can form an additive sequence.

A valid additive sequence should contain at least three numbers. Except for the first two numbers, each subsequent number in the sequence must be the sum of the preceding two.

Given a string containing only digits, return true if it is an additive number or false otherwise.

Note: Numbers in the additive sequence cannot have leading zeros, so sequence 1, 2, 03 or 1, 02, 3 is invalid.

Example 1:

Input: "112358"
Output: true
Explanation: The digits can form an additive sequence: 1, 1, 2, 3, 5, 8
1 + 1 = 2, 1 + 2 = 3, 2 + 3 = 5, 3 + 5 = 8

Example 2:

Input: "199100199"
Output: true
Explanation: The additive sequence is: 1, 99, 100, 199
1 + 99 = 100, 99 + 100 = 199

Constraints:

  • 1 <= num.length <= 35
  • num consists only of digits.

Solution

Python Solution

class Solution:
    def isAdditiveNumber(self, num: str) -> bool:
        n = len(num)
        for i in range(1, n//2 + 1):
            if num[0] == '0' and i > 1:
                break
            for j in range(1, (n-i)//2 + 1):
                if num[i] == '0' and j > 1:
                    break
                first = int(num[:i])
                second = int(num[i:i+j])
                start = i + j
                
                while start < n:
                    third = first + second
                    third_str = str(third)
                    third_len = len(third_str)
                    
                    if start + third_len > n or num[start:start+third_len] != third_str:
                        break
                        
                    first = second
                    second = third
                    start += third_len
                    
                if start == n:
                    return True
                    
        return False

Time Complexity: O(n³)

Where n is the length of the input string.

Space Complexity: O(1)

Only using a constant amount of extra space.

Java Solution

class Solution {
    public boolean isAdditiveNumber(String num) {
        int n = num.length();
        for (int i = 1; i <= n/2; i++) {
            if (num.charAt(0) == '0' && i > 1) break;
            for (int j = 1; j <= (n-i)/2; j++) {
                if (num.charAt(i) == '0' && j > 1) break;
                
                long first = Long.parseLong(num.substring(0, i));
                long second = Long.parseLong(num.substring(i, i+j));
                int start = i + j;
                
                if (isValid(num, first, second, start)) {
                    return true;
                }
            }
        }
        return false;
    }
    
    private boolean isValid(String num, long first, long second, int start) {
        if (start == num.length()) return true;
        
        long sum = first + second;
        String sumStr = String.valueOf(sum);
        if (start + sumStr.length() > num.length() || 
            !num.substring(start, start + sumStr.length()).equals(sumStr)) {
            return false;
        }
        
        return isValid(num, second, sum, start + sumStr.length());
    }
}

class Solution {
public:
    bool isAdditiveNumber(string num) {
        int n = num.length();
        for (int i = 1; i <= n/2; i++) {
            if (num[0] == '0' && i > 1) break;
            for (int j = 1; j <= (n-i)/2; j++) {
                if (num[i] == '0' && j > 1) break;
                
                string first = num.substr(0, i);
                string second = num.substr(i, j);
                int start = i + j;
                
                if (isValid(num, first, second, start)) {
                    return true;
                }
            }
        }
        return false;
    }
    
private:
    bool isValid(string& num, string first, string second, int start) {
        if (start == num.length()) return true;
        
        string sum = add(first, second);
        if (start + sum.length() > num.length() || 
            num.substr(start, sum.length()) != sum) {
            return false;
        }
        
        return isValid(num, second, sum, start + sum.length());
    }
    
    string add(string& a, string& b) {
        string res;
        int carry = 0;
        int i = a.length() - 1;
        int j = b.length() - 1;
        
        while (i >= 0 || j >= 0 || carry) {
            int sum = carry;
            if (i >= 0) sum += a[i--] - '0';
            if (j >= 0) sum += b[j--] - '0';
            carry = sum / 10;
            res = char(sum % 10 + '0') + res;
        }
        
        return res;
    }
};

/**
 * @param {string} num

var isAdditiveNumber = function(num) {
    const n = num.length;
    
    for (let i = 1; i <= n/2; i++) {
        if (num[0] === '0' && i > 1) break;
        for (let j = 1; j <= (n-i)/2; j++) {
            if (num[i] === '0' && j > 1) break;
            
            const first = num.slice(0, i);
            const second = num.slice(i, i+j);
            let start = i + j;
            
            let num1 = BigInt(first);
            let num2 = BigInt(second);
            
            while (start < n) {
                const sum = num1 + num2;
                const sumStr = sum.toString();
                const len = sumStr.length;
                
                if (start + len > n || num.slice(start, start + len) !== sumStr) {
                    break;
                }
                
                num1 = num2;
                num2 = sum;
                start += len;
            }
            
            if (start === n) return true;
        }
    }
    
    return false;
};

public class Solution {
    public bool IsAdditiveNumber(string num) {
        int n = num.Length;
        for (int i = 1; i <= n/2; i++) {
            if (num[0] == '0' && i > 1) break;
            for (int j = 1; j <= (n-i)/2; j++) {
                if (num[i] == '0' && j > 1) break;
                
                string first = num.Substring(0, i);
                string second = num.Substring(i, j);
                int start = i + j;
                
                if (IsValid(num, first, second, start)) {
                    return true;
                }
            }
        }
        return false;
    }
    
    private bool IsValid(string num, string first, string second, int start) {
        if (start == num.Length) return true;
        
        string sum = Add(first, second);
        if (start + sum.Length > num.Length || 
            num.Substring(start, sum.Length) != sum) {
            return false;
        }
        
        return IsValid(num, second, sum, start + sum.Length);
    }
    
    private string Add(string a, string b) {
        StringBuilder res = new StringBuilder();
        int carry = 0;
        int i = a.Length - 1;
        int j = b.Length - 1;
        
        while (i >= 0 || j >= 0 || carry > 0) {
            int sum = carry;
            if (i >= 0) sum += a[i--] - '0';
            if (j >= 0) sum += b[j--] - '0';
            carry = sum / 10;
            res.Insert(0, (char)(sum % 10 + '0'));
        }
        
        return res.ToString();
    }
}

Approach Explanation

The solution uses backtracking to find valid additive sequences:

  1. Key Insight:
    • Try all possible first two numbers
    • Check if sequence can be formed
    • Handle leading zeros
    • Handle large numbers
  2. Algorithm Steps:
    • Generate first two numbers
    • Validate remaining sequence
    • Check sum property
    • Handle edge cases

Example walkthrough:

  • Input: "112358"
  • First two numbers: 1, 1
  • Check sequence: 2, 3, 5, 8
  • Verify each sum

Optimization insights:

  • Early termination
  • Length constraints
  • Leading zero check
  • String arithmetic

Edge Cases:

  • Leading zeros
  • Large numbers
  • Short sequences
  • Invalid sequences
