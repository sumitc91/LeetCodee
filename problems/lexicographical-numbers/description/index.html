386. Lexicographical Numbers

Jump to Solution: Python Java C++ JavaScript C#

Problem Description

Given an integer n, return all the numbers in the range [1, n] sorted in lexicographical order.

You must write an algorithm that runs in O(n) time and uses O(1) extra space.

Example 1:

Input: n = 13
Output: [1,10,11,12,13,2,3,4,5,6,7,8,9]

Example 2:

Input: n = 2
Output: [1,2]

Constraints:

  • 1 <= n <= 5 * 10⁴

Solution

Python Solution

class Solution:
    def lexicalOrder(self, n: int) -> List[int]:
        result = []
        curr = 1
        
        for _ in range(n):
            result.append(curr)
            
            # Try to add a zero to current number
            if curr * 10 <= n:
                curr *= 10
            else:
                # If we can't add zero, increment the last digit
                while curr % 10 == 9 or curr + 1 > n:
                    curr //= 10
                if curr == 0:
                    break
                curr += 1
        
        return result

Time Complexity: O(n)

We generate exactly n numbers.

Space Complexity: O(1)

Excluding the output array, we only use constant extra space.

JavaScript Solution

/**
 * @param {number} n
 * @return {number[]}
 */
var lexicalOrder = function(n) {
    const result = [];
    let curr = 1;
    
    for (let i = 0; i < n; i++) {
        result.push(curr);
        
        // Try to add a zero to current number
        if (curr * 10 <= n) {
            curr *= 10;
        } else {
            // If we can't add zero, increment the last digit
            while (curr % 10 === 9 || curr + 1 > n) {
                curr = Math.floor(curr / 10);
            }
            if (curr === 0) {
                break;
            }
            curr++;
        }
    }
    
    return result;
};
