9. Palindrome Number

Easy

Problem Description

Given an integer x, return true if x is a palindrome, and false otherwise.

Examples

Example 1:
Input: x = 121
Output: true
Explanation: 121 reads as 121 from left to right and from right to left.

Example 2:
Input: x = -121
Output: false
Explanation: From left to right, it reads -121. From right to left, reads 121-. Therefore it is not a palindrome.

Example 3:
Input: x = 10
Output: false
Explanation: Reads 01 from right to left. Therefore it is not a palindrome.

Python Solution

class Solution:
    def isPalindrome(self, x: int) -> bool:
        if x < 0:
            return False
            
        original = x
        reversed_num = 0
        
        while x > 0:
            digit = x % 10
            reversed_num = reversed_num * 10 + digit
            x //= 10
            
        return original == reversed_num

JavaScript Solution

/**
 * @param {number} x
 * @return {boolean}
 */
var isPalindrome = function(x) {
    if (x < 0) {
        return false;
    }
    
    const original = x;
    let reversed = 0;
    
    while (x > 0) {
        const digit = x % 10;
        reversed = reversed * 10 + digit;
        x = Math.floor(x / 10);
    }
    
    return original === reversed;
};

Complexity Analysis

  • Time Complexity: O(log n) where n is the input number
  • Space Complexity: O(1) as we only use a constant amount of extra space

Solution Explanation

This solution uses a number reversal approach:

  • First, we handle negative numbers (they can't be palindromes)
  • We store the original number
  • We build the reversed number by:
    • Getting the last digit using modulo 10
    • Building the reversed number by multiplying by 10 and adding the digit
    • Removing the last digit by dividing by 10
  • Finally, we compare the original and reversed numbers

Key points:

  • We don't need to convert to string, making it more efficient
  • We handle edge cases (negative numbers, single digits) properly
  • The solution is space efficient as we only use a few variables
  • We can stop early if we detect it's not a palindrome
