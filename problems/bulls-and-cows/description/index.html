299. Bulls and Cows

Jump to Solution: Python Java C++ JavaScript C#

Problem Description

You are playing the Bulls and Cows game with your friend.

You write down a secret number and ask your friend to guess what the number is. When your friend makes a guess, you provide a hint with the following info:

  • The number of "bulls", which are digits in the guess that are in the correct position.
  • The number of "cows", which are digits in the guess that are in your secret number but are located in the wrong position.

Given the secret number secret and your friend's guess guess, return the hint for your friend's guess.

The hint should be formatted as "xAyB", where x is the number of bulls and y is the number of cows. Note that both secret and guess may contain duplicate digits.

Example 1:

Input: secret = "1807", guess = "7810"
Output: "1A3B"
Explanation: Bulls are connected with a '|' and cows are underlined:
- "1807" and "7810": 1 bull and 3 cows

Example 2:

Input: secret = "1123", guess = "0111"
Output: "1A1B"
Explanation: Bulls are connected with a '|' and cows are underlined:
- "1123" and "0111": 1 bull and 1 cow

Constraints:

  • 1 <= secret.length, guess.length <= 1000
  • secret.length == guess.length
  • secret and guess consist of digits only.

Solution

Python Solution

class Solution:
    def getHint(self, secret: str, guess: str) -> str:
        bulls = 0
        cows = 0
        secret_count = [0] * 10
        guess_count = [0] * 10
        
        # Count bulls and store remaining digits
        for s, g in zip(secret, guess):
            if s == g:
                bulls += 1
            else:
                secret_count[int(s)] += 1
                guess_count[int(g)] += 1
        
        # Count cows
        for i in range(10):
            cows += min(secret_count[i], guess_count[i])
            
        return f"{bulls}A{cows}B"

Time Complexity: O(n)

Where n is the length of the strings.

Space Complexity: O(1)

Using fixed size arrays for digit counts.

Java Solution

class Solution {
    public String getHint(String secret, String guess) {
        int bulls = 0;
        int cows = 0;
        int[] secretCount = new int[10];
        int[] guessCount = new int[10];
        
        // Count bulls and store remaining digits
        for (int i = 0; i < secret.length(); i++) {
            char s = secret.charAt(i);
            char g = guess.charAt(i);
            if (s == g) {
                bulls++;
            } else {
                secretCount[s - '0']++;
                guessCount[g - '0']++;
            }
        }
        
        // Count cows
        for (int i = 0; i < 10; i++) {
            cows += Math.min(secretCount[i], guessCount[i]);
        }
        
        return bulls + "A" + cows + "B";
    }
}

Approach Explanation

The solution uses counting arrays to track digits:

  1. Key Insight:
    • Count bulls first
    • Store remaining digits
    • Count common digits for cows
    • Handle duplicates correctly
  2. Algorithm Steps:
    • Compare positions for bulls
    • Count remaining digits
    • Find minimum counts for cows
    • Format result string

Example walkthrough:

  • "1807" vs "7810"
  • 8 matches position (1 bull)
  • 1,0,7 match elsewhere (3 cows)
  • Result: "1A3B"

Optimization insights:

  • Single pass for bulls
  • Fixed space complexity
  • Handles duplicates naturally
  • No sorting required

Edge Cases:

  • All bulls
  • All cows
  • No matches
  • Duplicate digits
