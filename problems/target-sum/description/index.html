494. Target Sum

Jump to Solution: Python Java C++ JavaScript C#

Problem Description

You are given an integer array nums and an integer target.

You want to build an expression out of nums by adding one of the symbols '+' and '-' before each integer in nums and then concatenate all the integers.

  • For example, if nums = [2, 1], you can add a '+' before 2 and a '-' before 1 and concatenate them to build the expression "+2-1".

Return the number of different expressions that you can build, which evaluates to target.

Example 1:

Input: nums = [1,1,1,1,1], target = 3
Output: 5
Explanation: There are 5 ways to assign symbols to make the sum of nums be target 3.
-1 + 1 + 1 + 1 + 1 = 3
+1 - 1 + 1 + 1 + 1 = 3
+1 + 1 - 1 + 1 + 1 = 3
+1 + 1 + 1 - 1 + 1 = 3
+1 + 1 + 1 + 1 - 1 = 3

Example 2:

Input: nums = [1], target = 1
Output: 1

Constraints:

  • 1 <= nums.length <= 20
  • 0 <= nums[i] <= 1000
  • 0 <= sum(nums[i]) <= 1000
  • -1000 <= target <= 1000

Solution

Python Solution

class Solution:
    def findTargetSumWays(self, nums: List[int], target: int) -> int:
        total = sum(nums)
        if abs(target) > total:
            return 0
        
        dp = {}
        
        def backtrack(index: int, curr_sum: int) -> int:
            if index == len(nums):
                return 1 if curr_sum == target else 0
            
            if (index, curr_sum) in dp:
                return dp[(index, curr_sum)]
            
            dp[(index, curr_sum)] = (
                backtrack(index + 1, curr_sum + nums[index]) +
                backtrack(index + 1, curr_sum - nums[index])
            )
            
            return dp[(index, curr_sum)]
        
        return backtrack(0, 0)

Time Complexity: O(n × total)

Where n is the length of nums and total is the sum of all numbers in nums.

Space Complexity: O(n × total)

For the memoization dictionary.

Java Solution

class Solution {
    private int[] nums;
    private int target;
    private Map memo;
    
    public int findTargetSumWays(int[] nums, int target) {
        this.nums = nums;
        this.target = target;
        this.memo = new HashMap<>();
        
        int total = 0;
        for (int num : nums) {
            total += num;
        }
        
        if (Math.abs(target) > total) {
            return 0;
        }
        
        return backtrack(0, 0);
    }
    
    private int backtrack(int index, int currSum) {
        if (index == nums.length) {
            return currSum == target ? 1 : 0;
        }
        
        String key = index + "," + currSum;
        if (memo.containsKey(key)) {
            return memo.get(key);
        }
        
        int result = backtrack(index + 1, currSum + nums[index]) +
                    backtrack(index + 1, currSum - nums[index]);
        
        memo.put(key, result);
        return result;
    }
}

/**
 * @param {number[]} nums
 * @param {number} target
 * @return {number}

var findTargetSumWays = function(nums, target) {
    const total = nums.reduce((a, b) => a + b, 0);
    if (Math.abs(target) > total) {
        return 0;
    }
    
    const memo = new Map();
    
    const backtrack = (index, currSum) => {
        if (index === nums.length) {
            return currSum === target ? 1 : 0;
        }
        
        const key = `${index},${currSum}`;
        if (memo.has(key)) {
            return memo.get(key);
        }
        
        const result = backtrack(index + 1, currSum + nums[index]) +
                      backtrack(index + 1, currSum - nums[index]);
        
        memo.set(key, result);
        return result;
    };
    
    return backtrack(0, 0);
};

C# Solution

public class Solution {
    private int[] nums;
    private int target;
    private Dictionary memo;
    
    public int FindTargetSumWays(int[] nums, int target) {
        this.nums = nums;
        this.target = target;
        this.memo = new Dictionary();
        
        int total = nums.Sum();
        if (Math.Abs(target) > total) {
            return 0;
        }
        
        return Backtrack(0, 0);
    }
    
    private int Backtrack(int index, int currSum) {
        if (index == nums.Length) {
            return currSum == target ? 1 : 0;
        }
        
        string key = $"{index},{currSum}";
        if (memo.ContainsKey(key)) {
            return memo[key];
        }
        
        int result = Backtrack(index + 1, currSum + nums[index]) +
                    Backtrack(index + 1, currSum - nums[index]);
        
        memo[key] = result;
        return result;
    }
}
