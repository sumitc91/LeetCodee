187. Repeated DNA Sequences

Medium

Problem Description

The DNA sequence is composed of a series of nucleotides abbreviated as 'A', 'C', 'G', and 'T'.

  • For example, "ACGAATTCCG" is a DNA sequence.

When studying DNA, it is useful to identify repeated sequences within the DNA.

Given a string s that represents a DNA sequence, return all the 10-letter-long sequences (substrings) that occur more than once in a DNA molecule. You may return the answer in any order.

Examples

Example 1:
Input: s = "AAAAACCCCCAAAAACCCCCCAAAAAGGGTTT"
Output: ["AAAAACCCCC","CCCCCAAAAA"]

Example 2:
Input: s = "AAAAAAAAAAAAA"
Output: ["AAAAAAAAAA"]

Python Solution


def findRepeatedDnaSequences(s: str) -> List[str]:
    seen = set()
    result = set()
    
    for i in range(len(s) - 9):
        curr = s[i:i+10]
        if curr in seen:
            result.add(curr)
        seen.add(curr)
    
    return list(result)

Java Solution


class Solution {
    public List findRepeatedDnaSequences(String s) {
        Set seen = new HashSet<>();
        Set result = new HashSet<>();
        
        for (int i = 0; i <= s.length() - 10; i++) {
            String curr = s.substring(i, i + 10);
            if (!seen.add(curr)) {
                result.add(curr);
            }
        }
        
        return new ArrayList<>(result);
    }
}

C++ Solution


class Solution {
public:
    vector findRepeatedDnaSequences(string s) {
        unordered_set seen;
        unordered_set result;
        
        for (int i = 0; i <= (int)s.length() - 10; i++) {
            string curr = s.substr(i, 10);
            if (seen.count(curr)) {
                result.insert(curr);
            }
            seen.insert(curr);
        }
        
        return vector(result.begin(), result.end());
    }
};

JavaScript Solution


/**
 * @param {string} s
 * @return {string[]}
 */
var findRepeatedDnaSequences = function(s) {
    const seen = new Set();
    const result = new Set();
    
    for (let i = 0; i <= s.length - 10; i++) {
        const curr = s.slice(i, i + 10);
        if (seen.has(curr)) {
            result.add(curr);
        }
        seen.add(curr);
    }
    
    return Array.from(result);
};

C# Solution


public class Solution {
    public IList FindRepeatedDnaSequences(string s) {
        var seen = new HashSet();
        var result = new HashSet();
        
        for (int i = 0; i <= s.Length - 10; i++) {

Complexity Analysis

  • Time Complexity: O(n) where n is the length of the input string
  • Space Complexity: O(n) to store the sequences in the sets

Solution Explanation

This solution uses hash sets to track sequences:

  • Key concept:
    • Sliding window
    • Hash set tracking
    • Duplicate detection
  • Algorithm steps:
    • Extract sequences
    • Track seen sequences
    • Record duplicates
    • Convert to list

Key points:

  • Fixed length (10)
  • String slicing
  • Set operations
  • Memory efficiency
