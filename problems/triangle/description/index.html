120. Triangle

Medium

Problem Description

Given a triangle array, return the minimum path sum from top to bottom.

For each step, you may move to an adjacent number of the row below. More formally, if you are on index i on the current row, you may move to either index i or index i + 1 on the next row.

Examples

Example 1:
Input: triangle = [[2],[3,4],[6,5,7],[4,1,8,3]]
Output: 11
Explanation: The triangle looks like:
   2
  3 4
 6 5 7
4 1 8 3
The minimum path sum from top to bottom is: 2 + 3 + 5 + 1 = 11

Example 2:
Input: triangle = [[-10]]
Output: -10

Python Solution


def minimumTotal(triangle: List[List[int]]) -> int:
    # Start from the second last row
    for i in range(len(triangle)-2, -1, -1):
        for j in range(len(triangle[i])):
            # Add minimum of two possible paths
            triangle[i][j] += min(triangle[i+1][j], triangle[i+1][j+1])
    
    return triangle[0][0]

Java Solution


class Solution {
    public int minimumTotal(List> triangle) {
        // Start from the second last row
        for (int i = triangle.size()-2; i >= 0; i--) {
            for (int j = 0; j < triangle.get(i).size(); j++) {
                // Add minimum of two possible paths
                int sum = triangle.get(i).get(j) + 
                         Math.min(triangle.get(i+1).get(j), 
                                triangle.get(i+1).get(j+1));
                triangle.get(i).set(j, sum);
            }
        }
        
        return triangle.get(0).get(0);
    }
}

    public int MinimumTotal(IList> triangle) {
        // Start from the second last row
        for (int i = triangle.Count-2; i >= 0; i--) {
            for (int j = 0; j < triangle[i].Count; j++) {

Solution Explanation

This solution uses bottom-up dynamic programming:

  • Key concept:
    • Bottom-up approach
    • In-place modification
    • Path minimization
  • Algorithm steps:
    • Start from bottom
    • Update each element
    • Choose minimum path
    • Propagate upwards

Key points:

  • Space optimization
  • In-place updates
  • Bottom-up DP
  • Path tracking
