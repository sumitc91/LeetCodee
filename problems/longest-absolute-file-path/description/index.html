388. Longest Absolute File Path

Jump to Solution: Python Java C++ JavaScript C#

Problem Description

Suppose we have a file system that stores both files and directories. An example of one system is represented in the following way: 

"dir\n\tsubdir1\n\t\tfile1.ext\n\t\tsubsubdir1\n\tsubdir2\n\t\tsubsubdir2\n\t\t\tfile2.ext"

Here, we have \n represents the newline character and \t represents a tab. The directory dir contains an empty sub-directory subdir1 and a sub-directory subdir2 containing a file file2.ext.

The string "dir\n\tsubdir1\n\t\tfile1.ext\n\t\tsubsubdir1\n\tsubdir2\n\t\tsubsubdir2\n\t\t\tfile2.ext" represents: 

dir
    subdir1
        file1.ext
        subsubdir1
    subdir2
        subsubdir2
            file2.ext

The directory dir contains two sub-directories subdir1 and subdir2. subdir1 contains a file file1.ext and an empty second-level sub-directory subsubdir1. subdir2 contains a second-level sub-directory subsubdir2 containing a file file2.ext.

We are interested in finding the longest (number of characters) absolute path to a file within our file system. For example, in the second example above, the longest absolute path is "dir/subdir2/subsubdir2/file2.ext", and its length is 32 (not including the double quotes).

Given a string input representing the file system in the explained format, return the length of the longest absolute path to a file in the abstracted file system. If there is no file in the system, return 0.

Note that the absolute path must end with a file name.

Example 1:

Input: input = "dir\n\tsubdir1\n\t\tfile1.ext\n\t\tsubsubdir1\n\tsubdir2\n\t\tsubsubdir2\n\t\t\tfile2.ext"
Output: 32
Explanation: We have two files:
"dir/subdir1/file1.ext" of length 21
"dir/subdir2/subsubdir2/file2.ext" of length 32
The longest absolute path is "dir/subdir2/subsubdir2/file2.ext", which has length 32.

Example 2:

Input: input = "dir\n\tsubdir1\n\t\tfile1.ext"
Output: 21
Explanation: The longest absolute path is "dir/subdir1/file1.ext", which has length 21.

Constraints:

  • 1 <= input.length <= 10⁴
  • input may contain lowercase or uppercase English letters, a newline character \n, a tab character \t, a dot ., a space or digits.

Solution

Python Solution

class Solution:
    def lengthLongestPath(self, input: str) -> int:
        # Split input into lines
        paths = input.split('\n')
        stack = []  # Stack to store path lengths
        max_len = 0
        
        for path in paths:
            # Count number of tabs to determine depth
            depth = path.count('\t')
            name = path.replace('\t', '')
            
            # Pop from stack until we're at the parent directory
            while len(stack) > depth:
                stack.pop()
            
            # Calculate current path length
            curr_len = (stack[-1] if stack else 0) + len(name)
            
            # If it's a file, update max_len
            if '.' in name:
                max_len = max(max_len, curr_len + depth)  # Add depth for '/' characters
            else:
                stack.append(curr_len)
        
        return max_len

Time Complexity: O(n)

Where n is the length of the input string. We process each character once.

Space Complexity: O(d)

Where d is the maximum depth of the directory structure.

Java Solution

class Solution {
    public int lengthLongestPath(String input) {
        // Split input into lines
        String[] paths = input.split("\n");
        Stack stack = new Stack<>();  // Stack to store path lengths
        int maxLen = 0;
        
        for (String path : paths) {
            // Count number of tabs to determine depth
            int depth = 0;
            while (depth < path.length() && path.charAt(depth) == '\t') {
                depth++;
            }
            
            // Get name without tabs
            String name = path.substring(depth);
            
            // Pop from stack until we're at the parent directory
            while (stack.size() > depth) {
                stack.pop();
            }
            
            // Calculate current path length
            int currLen = (stack.isEmpty() ? 0 : stack.peek()) + name.length();
            
            // If it's a file, update maxLen
            if (name.contains(".")) {
                maxLen = Math.max(maxLen, currLen + depth);  // Add depth for '/' characters
            } else {
                stack.push(currLen);
            }
        }
        
        return maxLen;
    }
}

public class Solution {
    public int LengthLongestPath(string input) {
        // Split input into lines
        string[] paths = input.Split('\n');
        Stack stack = new Stack();  // Stack to store path lengths
        int maxLen = 0;
        
        foreach (string path in paths) {
            // Count number of tabs to determine depth
            int depth = 0;
            while (depth < path.Length && path[depth] == '\t') {
                depth++;
            }
            
            // Get name without tabs
            string name = path.Substring(depth);
            
            // Pop from stack until we're at the parent directory
            while (stack.Count > depth) {
                stack.Pop();
            }
            
            // Calculate current path length
            int currLen = (stack.Count > 0 ? stack.Peek() : 0) + name.Length;
            
            // If it's a file, update maxLen
            if (name.Contains(".")) {
                maxLen = Math.Max(maxLen, currLen + depth);  // Add depth for '/' characters
            } else {
                stack.Push(currLen);
            }
        }
        
        return maxLen;
    }
}
