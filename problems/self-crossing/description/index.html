335. Self Crossing

Jump to Solution: Python Java C++ JavaScript C#

Problem Description

You are given an array of integers distance. You start at point (0,0) on a 2D plane and you move distance[0] steps with direction North, then distance[1] steps with direction West, distance[2] steps with direction South, distance[3] steps with direction East, and so on (counter-clockwise direction).

Return true if your path crosses itself, and false if it does not.

Example 1:

Input: distance = [2,1,1,2]
Output: true

Example 2:

Input: distance = [1,2,3,4]
Output: false

Example 3:

Input: distance = [1,1,1,1]
Output: true

Constraints:

  • 1 <= distance.length <= 10⁵
  • 1 <= distance[i] <= 10⁵

Solution

Python Solution

class Solution:
    def isSelfCrossing(self, distance: List[int]) -> bool:
        n = len(distance)
        if n < 4:
            return False
            
        for i in range(3, n):
            # Case 1: Current line crosses the line 3 steps before
            if distance[i] >= distance[i-2] and distance[i-1] <= distance[i-3]:
                return True
                
            # Case 2: Current line crosses the line 4 steps before
            if i >= 4 and distance[i-1] == distance[i-3] and distance[i] + distance[i-4] >= distance[i-2]:
                return True
                
            # Case 3: Current line crosses the line 5 steps before
            if i >= 5 and distance[i-2] >= distance[i-4] and distance[i] + distance[i-4] >= distance[i-2] and \
               distance[i-1] <= distance[i-3] and distance[i-1] + distance[i-5] >= distance[i-3]:
                return True
                
        return False
