123. Best Time to Buy and Sell Stock III

Hard

Problem Description

You are given an array prices where prices[i] is the price of a given stock on the ith day.

Find the maximum profit you can achieve. You may complete at most two transactions.

Note: You may not engage in multiple transactions simultaneously (i.e., you must sell the stock before you buy again).

Examples

Example 1:
Input: prices = [3,3,5,0,0,3,1,4]
Output: 6
Explanation: Buy on day 4 (price = 0) and sell on day 6 (price = 3), profit = 3-0 = 3.
Then buy on day 7 (price = 1) and sell on day 8 (price = 4), profit = 4-1 = 3.

Example 2:
Input: prices = [1,2,3,4,5]
Output: 4
Explanation: Buy on day 1 (price = 1) and sell on day 5 (price = 5), profit = 5-1 = 4.
Note that you cannot buy on day 1, buy on day 2 and sell them later, as you are engaging in multiple transactions at the same time. You must sell before buying again.

Example 3:
Input: prices = [7,6,4,3,1]
Output: 0
Explanation: In this case, no transaction is done, i.e. max profit = 0.

Python Solution


def maxProfit(prices: List[int]) -> int:
    if not prices:
        return 0
    
    # Initialize states
    buy1 = float('inf')  # First buy price
    buy2 = float('inf')  # Second buy price
    profit1 = 0  # Profit after first transaction
    profit2 = 0  # Profit after second transaction
    
    for price in prices:
        # Update the states
        buy1 = min(buy1, price)  # Minimize first buy price
        profit1 = max(profit1, price - buy1)  # Maximize first profit
        buy2 = min(buy2, price - profit1)  # Minimize second buy price considering first profit
        profit2 = max(profit2, price - buy2)  # Maximize second profit
    
    return profit2

Java Solution


class Solution {
    public int maxProfit(int[] prices) {
        if (prices == null || prices.length == 0) {
            return 0;
        }
        
        // Initialize states
        int buy1 = Integer.MAX_VALUE;  // First buy price
        int buy2 = Integer.MAX_VALUE;  // Second buy price
        int profit1 = 0;  // Profit after first transaction
        int profit2 = 0;  // Profit after second transaction
        
        for (int price : prices) {
            // Update the states
            buy1 = Math.min(buy1, price);  // Minimize first buy price
            profit1 = Math.max(profit1, price - buy1);  // Maximize first profit
            buy2 = Math.min(buy2, price - profit1);  // Minimize second buy price
            profit2 = Math.max(profit2, price - buy2);  // Maximize second profit
        }
        
        return profit2;
    }
}

C# Solution


public class Solution {
    public int MaxProfit(int[] prices) {
        if (prices == null || prices.Length == 0) {
            return 0;
        }
        
        // Initialize states
        int buy1 = int.MaxValue;  // First buy price
        int buy2 = int.MaxValue;  // Second buy price
        int profit1 = 0;  // Profit after first transaction
        int profit2 = 0;  // Profit after second transaction
        
        foreach (int price in prices) {
            // Update the states
            buy1 = Math.Min(buy1, price);  // Minimize first buy price
            profit1 = Math.Max(profit1, price - buy1);  // Maximize first profit
            buy2 = Math.Min(buy2, price - profit1);  // Minimize second buy price
            profit2 = Math.Max(profit2, price - buy2);  // Maximize second profit
        }
        
        return profit2;
    }
}

Complexity Analysis

  • Time Complexity: O(n) where n is the length of the prices array
  • Space Complexity: O(1) as we only use four variables

Solution Explanation

This solution uses a state machine approach:

  • Key concept:
    • Four states tracking
    • State transitions
    • Profit maximization
  • Algorithm steps:
    • Track buy prices
    • Track profits
    • Update states
    • Maximize profits

Key points:

  • Single pass solution
  • Constant space
  • State optimization
  • Handle edge cases
