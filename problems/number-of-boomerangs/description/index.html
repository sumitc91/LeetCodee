447. Number of Boomerangs

Jump to Solution: Python Java C++ JavaScript C#

Problem Description

You are given n points in the plane that are all distinct, where points[i] = [xi, yi]. A boomerang is a tuple of points (i, j, k) such that the distance between i and j equals the distance between i and k (the order of the tuple matters).

Return the number of boomerangs.

Example 1:

Input: points = [[0,0],[1,0],[2,0]]
Output: 2
Explanation: The two boomerangs are [[1,0],[0,0],[2,0]] and [[1,0],[2,0],[0,0]].

Example 2:

Input: points = [[1,1],[2,2],[3,3]]
Output: 2

Example 3:

Input: points = [[1,1]]
Output: 0

Constraints:

  • n == points.length
  • 1 <= n <= 500
  • points[i].length == 2
  • -10⁴ <= xi, yi <= 10⁴
  • All the points are unique.

Solution

Python Solution

class Solution:
    def numberOfBoomerangs(self, points: List[List[int]]) -> int:
        result = 0
        
        for p1 in points:
            # Store distances from p1 to all other points
            distances = {}
            
            for p2 in points:
                if p1 != p2:
                    # Calculate squared distance to avoid floating point precision issues
                    dx = p1[0] - p2[0]
                    dy = p1[1] - p2[1]
                    dist = dx * dx + dy * dy
                    
                    # Count points at this distance
                    distances[dist] = distances.get(dist, 0) + 1
            
            # For each distance, calculate number of boomerangs
            for count in distances.values():
                # If we have n points at same distance, we can make n * (n-1) boomerangs
                result += count * (count - 1)
        
        return result

C# Solution

public class Solution {
    public int NumberOfBoomerangs(int[][] points) {
        int result = 0;
        
        foreach (var p1 in points) {
            // Store distances from p1 to all other points
            Dictionary distances = new Dictionary();
            
            foreach (var p2 in points) {
                if (p1 != p2) {
                    // Calculate squared distance to avoid floating point precision issues
                    int dx = p1[0] - p2[0];
                    int dy = p1[1] - p2[1];
                    int dist = dx * dx + dy * dy;
                    
                    // Count points at this distance
                    if (!distances.ContainsKey(dist)) {
                        distances[dist] = 0;
                    }
                    distances[dist]++;
                }
            }
            
            // For each distance, calculate number of boomerangs
            foreach (var count in distances.Values) {
                // If we have n points at same distance, we can make n * (n-1) boomerangs
                result += count * (count - 1);
            }
        }
        
        return result;
    }
}
