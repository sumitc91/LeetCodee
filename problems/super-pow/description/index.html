372. Super Pow

Jump to Solution: Python Java C++ JavaScript C#

Problem Description

Your task is to calculate a^b mod 1337 where a is a positive integer and b is an extremely large positive integer given in the form of an array.

Example 1:

Input: a = 2, b = [3]
Output: 8

Example 2:

Input: a = 2, b = [1,0]
Output: 1024

Example 3:

Input: a = 1, b = [4,3,3,8,5,2]
Output: 1

Constraints:

  • 1 <= a <= 2³¹ - 1
  • 1 <= b.length <= 2000
  • 0 <= b[i] <= 9
  • b doesn't contain leading zeros.

Solution

Python Solution

class Solution:
    def superPow(self, a: int, b: List[int]) -> int:
        MOD = 1337
        
        def powmod(x: int, n: int) -> int:
            # Calculate (x^n) % MOD
            result = 1
            x %= MOD
            while n > 0:
                if n & 1:
                    result = (result * x) % MOD
                x = (x * x) % MOD
                n >>= 1
            return result
        
        # Handle base case
        if not b:
            return 1
        
        # Use the property: (a^b) % m = ((a % m)^b) % m
        a %= MOD
        
        # Process each digit from left to right
        result = 1
        for digit in b:
            # For each new digit, we need to:
            # 1. Raise previous result to power of 10
            # 2. Multiply by a^digit
            result = (powmod(result, 10) * powmod(a, digit)) % MOD
        
        return result

Time Complexity: O(n)

Where n is the length of array b. Each powmod operation is O(log k) where k is the exponent.

Space Complexity: O(1)

Only uses a constant amount of extra space.
