470. Implement Rand10() Using Rand7()

Jump to Solution: Python Java C++ JavaScript C#

Problem Description

Given the API rand7() that generates a uniform random integer in the range [1, 7], write a function rand10() that generates a uniform random integer in the range [1, 10]. You can only call the API rand7(), and you are not allowed to predeclare any arrays or variables in your solution.

Each test case will have one or more calls to rand10(), and there will be at least one call to rand10() per test case.

Example 1:

Input: n = 1
Output: [2]

Example 2:

Input: n = 2
Output: [2,8]

Example 3:

Input: n = 3
Output: [3,8,10]

Constraints:

  • 1 <= n <= 10⁵

Solution

Python Solution

class Solution:
    def rand10(self):
        """
        :rtype: int
        """
        while True:
            # Generate a number from 1 to 49 with uniform probability
            row = rand7()
            col = rand7()
            idx = (row - 1) * 7 + col  # 1 to 49
            
            # If idx > 40, reject and try again
            # We only use first 40 numbers to ensure uniformity
            if idx <= 40:
                return 1 + (idx - 1) % 10

Time Complexity: O(1) average, O(∞) worst case

The probability of rejection is low, making the average number of calls to rand7() close to constant.

Space Complexity: O(1)

Only constant extra space is used.
