371. Sum of Two Integers

Jump to Solution: Python Java C++ JavaScript C#

Problem Description

Given two integers a and b, return the sum of the two integers without using the operators + and -.

Example 1:

Input: a = 1, b = 2
Output: 3

Example 2:

Input: a = 2, b = 3
Output: 5

Constraints:

  • -1000 <= a, b <= 1000

Solution

Python Solution

class Solution:
    def getSum(self, a: int, b: int) -> int:
        # Python has unlimited integer range, need to handle overflow manually
        mask = 0xffffffff
        
        # Convert a and b to 32-bit integers
        a = a & mask
        b = b & mask
        
        while b != 0:
            # Calculate carry
            carry = (a & b) << 1
            # Calculate sum without carry
            a = (a ^ b) & mask
            b = carry & mask
        
        # Handle negative numbers
        if a > 0x7fffffff:
            return ~(a ^ mask)
        return a

Time Complexity: O(1)

The number of iterations is bounded by the number of bits.

Space Complexity: O(1)

Only uses a constant amount of extra space.

Java Solution

class Solution {
    public int getSum(int a, int b) {
        while (b != 0) {
            // Calculate carry
            int carry = a & b;
            // Calculate sum without carry
            a = a ^ b;
            // Shift carry to next position
            b = carry << 1;
        }
        return a;
    }
}

Approach Explanation

The solution uses bitwise operations to simulate addition:

  1. Key Insight:
    • XOR for sum without carry
    • AND for carry bits
    • Left shift for carry position
    • Iterate until no carry
  2. Bitwise Operations:
    • XOR (^): adds bits without carry
    • AND (&): identifies carry bits
    • Left shift (<<): moves carry to next position
    • Overflow handling
