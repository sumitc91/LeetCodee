396. Rotate Function

Jump to Solution: Python Java C++ JavaScript C#

Problem Description

You are given an integer array nums of length n.

Assume arrk to be an array obtained by rotating nums by k positions clock-wise. We define the rotation function F on nums as follow:

  • F(k) = 0 * arrk[0] + 1 * arrk[1] + ... + (n-1) * arrk[n-1]

Return the maximum value of F(0), F(1), ..., F(n-1).

Example 1:

Input: nums = [4,3,2,6]
Output: 26
Explanation:
F(0) = (0 * 4) + (1 * 3) + (2 * 2) + (3 * 6) = 0 + 3 + 4 + 18 = 25
F(1) = (0 * 6) + (1 * 4) + (2 * 3) + (3 * 2) = 0 + 4 + 6 + 6 = 16
F(2) = (0 * 2) + (1 * 6) + (2 * 4) + (3 * 3) = 0 + 6 + 8 + 9 = 23
F(3) = (0 * 3) + (1 * 2) + (2 * 6) + (3 * 4) = 0 + 2 + 12 + 12 = 26
Therefore, F(3) = 26 is the maximum value.

Constraints:

  • n == nums.length
  • 1 <= n <= 10⁵
  • -100 <= nums[i] <= 100

Solution

Python Solution

class Solution:
    def maxRotateFunction(self, nums: List[int]) -> int:
        # Calculate initial F(0)
        n = len(nums)
        array_sum = sum(nums)
        curr_sum = sum(i * num for i, num in enumerate(nums))
        max_sum = curr_sum
        
        # Calculate F(k) using the formula:
        # F(k) = F(k-1) + sum(nums) - n * nums[n-k]
        for i in range(1, n):
            curr_sum = curr_sum + array_sum - n * nums[n-i]
            max_sum = max(max_sum, curr_sum)
        
        return max_sum

Time Complexity: O(n)

Where n is the length of the array. We only need one pass through the array.

Space Complexity: O(1)

We only use a constant amount of extra space.

        }

    }
}

public class Solution {
    public int MaxRotateFunction(int[] nums) {
        // Calculate initial F(0)
        int n = nums.Length;
        int arraySum = nums.Sum();
        int currSum = 0;
        
        for (int i = 0; i < n; i++) {
            currSum += i * nums[i];
        }
        
        int maxSum = currSum;
        
        // Calculate F(k) using the formula:
        // F(k) = F(k-1) + sum(nums) - n * nums[n-k]
        for (int i = 1; i < n; i++) {
            currSum = currSum + arraySum - n * nums[n-i];
            maxSum = Math.Max(maxSum, currSum);
        }
        
        return maxSum;
    }
}
