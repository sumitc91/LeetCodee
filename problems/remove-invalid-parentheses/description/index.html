301. Remove Invalid Parentheses

Jump to Solution: Python Java C++ JavaScript C#

Problem Description

Given a string s that contains parentheses and letters, remove the minimum number of invalid parentheses to make the input string valid.

Return all the possible results. You may return the answer in any order.

Example 1:

Input: s = "()())()"
Output: ["(())()","()()()"]

Example 2:

Input: s = "(a)())()"
Output: ["(a())()","(a)()()"]

Example 3:

Input: s = ")("
Output: [""]

Constraints:

  • 1 <= s.length <= 25
  • s consists of lowercase English letters and parentheses '(' and ')'.
  • There will be at most 20 parentheses in s.

Solution

Python Solution

class Solution:
    def removeInvalidParentheses(self, s: str) -> List[str]:
        def isValid(s):
            count = 0
            for char in s:
                if char == '(':
                    count += 1
                elif char == ')':
                    count -= 1
                if count < 0:
                    return False
            return count == 0
        
        def backtrack(s, start, left_rem, right_rem, result):
            if left_rem == 0 and right_rem == 0:
                if isValid(s):
                    result.add(s)
                return
            
            for i in range(start, len(s)):
                if i > start and s[i] == s[i-1]:
                    continue
                    
                if right_rem > 0 and s[i] == ')':
                    backtrack(s[:i] + s[i+1:], i, left_rem, right_rem-1, result)
                elif left_rem > 0 and s[i] == '(':
                    backtrack(s[:i] + s[i+1:], i, left_rem-1, right_rem, result)
        
        left_rem = right_rem = 0
        for char in s:
            if char == '(':
                left_rem += 1
            elif char == ')':
                if left_rem > 0:
                    left_rem -= 1
                else:
                    right_rem += 1
                    
        result = set()
        backtrack(s, 0, left_rem, right_rem, result)
        return list(result) if result else [""]

Time Complexity: O(2^n)

In the worst case, we need to try all possible combinations.

Space Complexity: O(n)

For recursion stack and storing results.

Java Solution

class Solution {
    public List removeInvalidParentheses(String s) {
        List result = new ArrayList<>();
        if (s == null) return result;
        
        Set visited = new HashSet<>();
        Queue queue = new LinkedList<>();
        
        queue.add(s);
        visited.add(s);
        
        boolean found = false;
        
        while (!queue.isEmpty()) {
            s = queue.poll();
            
            if (isValid(s)) {
                result.add(s);
                found = true;
            }
            
            if (found) continue;
            
            for (int i = 0; i < s.length(); i++) {
                if (s.charAt(i) != '(' && s.charAt(i) != ')') continue;
                
                String t = s.substring(0, i) + s.substring(i + 1);
                
                if (!visited.contains(t)) {
                    queue.add(t);
                    visited.add(t);
                }
            }
        }
        
        return result.isEmpty() ? Arrays.asList("") : result;
    }
    
    private boolean isValid(String s) {
        int count = 0;
        
        for (char c : s.toCharArray()) {
            if (c == '(') count++;
            if (c == ')') count--;
            if (count < 0) return false;
        }
        
        return count == 0;
    }
}

JavaScript Solution

/**
 * @param {string} s
 * @return {string[]}
 */
var removeInvalidParentheses = function(s) {
    const isValid = (s) => {
        let count = 0;
        for (let char of s) {
            if (char === '(') count++;
            else if (char === ')') count--;
            if (count < 0) return false;
        }
        return count === 0;
    };
    
    const result = [];
    const visited = new Set();
    const queue = [s];
    visited.add(s);
    let found = false;
    
    while (queue.length > 0) {
        s = queue.shift();
        
        if (isValid(s)) {
            result.push(s);
            found = true;
        }
        
        if (found) continue;
        
        for (let i = 0; i < s.length; i++) {
            if (s[i] !== '(' && s[i] !== ')') continue;
            
            const t = s.slice(0, i) + s.slice(i + 1);
            
            if (!visited.has(t)) {
                queue.push(t);
                visited.add(t);
            }
        }
    }
    
    return result.length ? result : [""];
};

C# Solution

public class Solution {
    public IList RemoveInvalidParentheses(string s) {
        var result = new List();
        if (string.IsNullOrEmpty(s)) {
            result.Add("");
            return result;
        }
        
        var visited = new HashSet();
        var queue = new Queue();
        
        queue.Enqueue(s);
        visited.Add(s);
        bool found = false;
        
        while (queue.Count > 0) {
            s = queue.Dequeue();
            
            if (IsValid(s)) {
                result.Add(s);
                found = true;
            }
            
            if (found) continue;
            
            for (int i = 0; i < s.Length; i++) {
                if (s[i] != '(' && s[i] != ')') continue;
                
                string t = s.Substring(0, i) + s.Substring(i + 1);
                
                if (!visited.Contains(t)) {
                    queue.Enqueue(t);
                    visited.Add(t);
                }
            }
        }
        
        return result.Count > 0 ? result : new List { "" };
    }
    
    private bool IsValid(string s) {
        int count = 0;
        
        foreach (char c in s) {
            if (c == '(') count++;
            if (c == ')') count--;
            if (count < 0) return false;
        }
        
        return count == 0;
    }
}

Approach Explanation

The solution uses BFS/DFS to find all valid combinations:

  1. Key Insight:
    • Remove one parenthesis at a time
    • Check validity at each step
    • Stop when valid strings found
    • Use visited set for optimization
  2. Algorithm Steps:
    • Count mismatched parentheses
    • Try removing each parenthesis
    • Check string validity
    • Collect all valid results

Example walkthrough:

  • Input: "()())()"
  • Remove one ')'
  • Check each resulting string
  • Find all valid combinations

Optimization insights:

  • Early termination
  • Skip duplicate removals
  • Level-wise BFS
  • Visited set for pruning

Edge Cases:

  • Empty string
  • All invalid parentheses
  • Already valid string
  • Multiple valid solutions
