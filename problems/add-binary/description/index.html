67. Add Binary

Easy

Problem Description

Given two binary strings a and b, return their sum as a binary string.

Examples

Example 1:
Input: a = "11", b = "1"
Output: "100"

Example 2:
Input: a = "1010", b = "1011"
Output: "10101"

Python Solution


def addBinary(a: str, b: str) -> str:
    result = []
    carry = 0
    i = len(a) - 1
    j = len(b) - 1
    
    while i >= 0 or j >= 0 or carry:
        # Get digits from right to left
        digit_a = int(a[i]) if i >= 0 else 0
        digit_b = int(b[j]) if j >= 0 else 0
        
        # Calculate current sum and carry
        current_sum = digit_a + digit_b + carry
        carry = current_sum // 2
        result.append(str(current_sum % 2))
        
        i -= 1
        j -= 1
    
    return ''.join(result[::-1])

Java Solution


class Solution {
    public String addBinary(String a, String b) {
        StringBuilder result = new StringBuilder();
        int carry = 0;
        int i = a.length() - 1;
        int j = b.length() - 1;
        
        while (i >= 0 || j >= 0 || carry > 0) {
            // Get digits from right to left
            int digitA = i >= 0 ? a.charAt(i) - '0' : 0;
            int digitB = j >= 0 ? b.charAt(j) - '0' : 0;
            
            // Calculate current sum and carry
            int currentSum = digitA + digitB + carry;
            carry = currentSum / 2;
            result.append(currentSum % 2);
            
            i--;
            j--;
        }
        
        return result.reverse().toString();
    }
}

JavaScript Solution


/**
 * @param {string} a
 * @param {string} b
 * @return {string}
 */
var addBinary = function(a, b) {
    let result = '';
    let carry = 0;
    let i = a.length - 1;
    let j = b.length - 1;
    
    while (i >= 0 || j >= 0 || carry > 0) {
        // Get digits from right to left
        const digitA = i >= 0 ? parseInt(a[i]) : 0;
        const digitB = j >= 0 ? parseInt(b[j]) : 0;
        
        // Calculate current sum and carry
        const currentSum = digitA + digitB + carry;
        carry = Math.floor(currentSum / 2);
        result = (currentSum % 2) + result;
        
        i--;
        j--;
    }
    
    return result;
};

C# Solution


public class Solution {
    public string AddBinary(string a, string b) {
        StringBuilder result = new StringBuilder();
        int carry = 0;
        int i = a.Length - 1;
        int j = b.Length - 1;
        
        while (i >= 0 || j >= 0 || carry > 0) {
            // Get digits from right to left
            int digitA = i >= 0 ? a[i] - '0' : 0;
            int digitB = j >= 0 ? b[j] - '0' : 0;
            
            // Calculate current sum and carry
            int currentSum = digitA + digitB + carry;
            carry = currentSum / 2;
            result.Insert(0, currentSum % 2);
            
            i--;
            j--;
        }
        
        return result.ToString();
    }
}

Complexity Analysis

  • Time Complexity: O(max(N, M)) where N and M are lengths of input strings
  • Space Complexity: O(max(N, M)) to store the result

Solution Explanation

This solution implements binary addition with carry:

  • Key concept:
    • Process digits from right to left
    • Handle carry propagation
    • Build result string
  • Algorithm steps:
    • Initialize carry and result
    • Process digits while either:
      • Digits remain in either number
      • Carry remains to be processed
    • For each position:
      • Add digits and carry
      • Update carry
      • Append remainder to result

Key points:

  • Handle different length inputs
  • Process carry correctly
  • Build result efficiently
  • No need for binary conversion
