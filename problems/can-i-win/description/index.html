464. Can I Win

Jump to Solution: Python Java C++ JavaScript C#

Problem Description

In the "100 game" two players take turns adding, to a running total, any integer from 1 to 10. The player who first causes the running total to reach or exceed 100 wins.

What if we change the game so that players cannot re-use integers?

For example, two players might take turns drawing from a common pool of numbers from 1 to 15 without replacement until they reach a total >= 100.

Given two integers maxChoosableInteger and desiredTotal, return true if the first player to move can force a win, otherwise, return false. Assume both players play optimally.

Example 1:

Input: maxChoosableInteger = 10, desiredTotal = 11
Output: false
Explanation:
No matter which integer the first player choose, the first player will lose.
The first player can choose an integer from 1 up to 10.
If the first player choose 1, the second player can only choose integers from 2 up to 10.
The second player will win by choosing 10 and get a total = 11, which is >= desiredTotal.
Same with other integers chosen by the first player, the second player will always win.

Example 2:

Input: maxChoosableInteger = 10, desiredTotal = 0
Output: true

Example 3:

Input: maxChoosableInteger = 10, desiredTotal = 1
Output: true

Constraints:

  • 1 <= maxChoosableInteger <= 20
  • 0 <= desiredTotal <= 300

Solution

Python Solution

class Solution:
    def canIWin(self, maxChoosableInteger: int, desiredTotal: int) -> bool:
        # If sum of all numbers is less than desired total, no one can win
        if (maxChoosableInteger * (maxChoosableInteger + 1)) // 2 < desiredTotal:
            return False
        
        # If desired total is 0, first player wins
        if desiredTotal <= 0:
            return True
        
        self.memo = {}
        
        def can_win(choices, total):
            # Convert choices to tuple for hashing
            choices_tuple = tuple(choices)
            
            # If this state has been computed before, return the cached result
            if choices_tuple in self.memo:
                return self.memo[choices_tuple]
            
            # Try each available number
            for i in range(len(choices)):
                curr = choices[i]
                
                # If current number reaches desired total, player wins
                if curr >= total:
                    self.memo[choices_tuple] = True
                    return True
                
                # Make recursive call with remaining numbers
                remaining = choices[:i] + choices[i+1:]
                # If opponent can't win with remaining numbers and new total,
                # current player wins
                if not can_win(remaining, total - curr):
                    self.memo[choices_tuple] = True
                    return True
            
            # If no winning move found, current player loses
            self.memo[choices_tuple] = False
            return False
        
        # Start with all available numbers
        choices = list(range(1, maxChoosableInteger + 1))
        return can_win(choices, desiredTotal)

Java Solution

class Solution {
    private Map memo;
    
    public boolean canIWin(int maxChoosableInteger, int desiredTotal) {
        // If sum of all numbers is less than desired total, no one can win
        if ((maxChoosableInteger * (maxChoosableInteger + 1)) / 2 < desiredTotal) {
            return false;
        }
        
        // If desired total is 0, first player wins
        if (desiredTotal <= 0) {
            return true;
        }
        
        memo = new HashMap<>();
        return canWin(0, maxChoosableInteger, desiredTotal);
    }
    
    private boolean canWin(int state, int maxChoosableInteger, int total) {
        if (total <= 0) {
            return false;
        }
        
        if (memo.containsKey(state)) {
            return memo.get(state);
        }
        
        for (int i = 1; i <= maxChoosableInteger; i++) {
            int curr = 1 << (i - 1);
            if ((state & curr) == 0) {
                if (!canWin(state | curr, maxChoosableInteger, total - i)) {
                    memo.put(state, true);
                    return true;
                }
            }
        }
        
        memo.put(state, false);
        return false;
    }
}
