365. Water and Jug Problem

Jump to Solution: Python Java C++ JavaScript C#

Problem Description

You are given two jugs with capacities jug1Capacity and jug2Capacity liters. There is an infinite amount of water supply available. Determine whether it is possible to measure exactly targetCapacity liters using these two jugs.

Operations allowed:

  • Fill any of the jugs completely with water.
  • Empty any of the jugs.
  • Pour water from one jug into another till the other jug is completely full or the first jug itself is empty.

Example 1:

Input: jug1Capacity = 3, jug2Capacity = 5, targetCapacity = 4
Output: true
Explanation: The operations can be as follows:
1. Fill jug2 with 5 liters of water.
2. Pour 3 liters from jug2 to jug1.
3. Now jug2 has 2 liters and jug1 has 3 liters of water.
4. Empty jug1.
5. Pour 2 liters from jug2 to jug1.
6. Fill jug2 with 5 liters of water.
7. Pour 1 liter from jug2 to jug1.
Now jug1 has 3 liters and jug2 has 4 liters of water.

Example 2:

Input: jug1Capacity = 2, jug2Capacity = 6, targetCapacity = 5
Output: false

Example 3:

Input: jug1Capacity = 1, jug2Capacity = 2, targetCapacity = 3
Output: true

Constraints:

  • 1 <= jug1Capacity, jug2Capacity, targetCapacity <= 10⁶

Solution

Python Solution

class Solution:
    def canMeasureWater(self, jug1Capacity: int, jug2Capacity: int, targetCapacity: int) -> bool:
        # If target is greater than sum of both jugs, it's impossible
        if targetCapacity > jug1Capacity + jug2Capacity:
            return False
        
        # If target is 0, it's always possible
        if targetCapacity == 0:
            return True
        
        # Use Bézout's identity
        # Target must be a linear combination of jug capacities
        # i.e., target = x*jug1 + y*jug2 where x,y are integers
        # This is possible if and only if target is divisible by GCD of jug capacities
        def gcd(a: int, b: int) -> int:
            while b:
                a, b = b, a % b
            return a
        
        return targetCapacity % gcd(jug1Capacity, jug2Capacity) == 0

Time Complexity: O(log n)

Where n is the maximum of jug capacities (for GCD calculation).

Space Complexity: O(1)

Only uses a constant amount of extra space.

var canMeasureWater = function(jug1Capacity, jug2Capacity, targetCapacity) {
    // If target is greater than sum of both jugs, it's impossible

        return false;
    }

    if (targetCapacity === 0) {
        return true;

    const gcd = (a, b) => {
        while (b) {
            [a, b] = [b, a % b];
        }
        return a;
    };
    
    return targetCapacity % gcd(jug1Capacity, jug2Capacity) === 0;
};
