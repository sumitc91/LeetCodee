430. Flatten a Multilevel Doubly Linked List

Jump to Solution: Python Java C++ JavaScript C#

Problem Description

You are given a doubly linked list, which contains nodes that have a next pointer, a previous pointer, and an additional child pointer. This child pointer may or may not point to a separate doubly linked list, also containing these special nodes. These child lists may have one or more children of their own, and so on, to produce a multilevel data structure as shown in the example below.

Given the head of the first level of the list, flatten the list so that all the nodes appear in a single-level, doubly linked list. Let curr be a node with a child list. The nodes in the child list should appear after curr and before curr.next in the flattened list.

Return the head of the flattened list. The nodes in the list must have all of their child pointers set to null.

Example 1:

Input: head = [1,2,3,4,5,6,null,null,null,7,8,9,10,null,null,11,12]
Output: [1,2,3,7,8,11,12,9,10,4,5,6]

Example 2:

Input: head = [1,2,null,3]
Output: [1,3,2]
Explanation: The multilevel linked list in the input is shown.

Constraints:

  • The number of Nodes will not exceed 1000
  • 1 <= Node.val <= 10⁵
  • Node.prev is null or points to a node in the list
  • Node.next is null or points to a node in the list
  • Node.child is null or points to a node in a different list

Solution

Python Solution

class Solution:
    def flatten(self, head: 'Node') -> 'Node':
        if not head:
            return None
        
        # Create a dummy node to handle edge cases
        dummy = Node(0)
        curr = dummy
        stack = [head]
        
        while stack:
            node = stack.pop()
            
            # Add next node to stack if exists
            if node.next:
                stack.append(node.next)
            
            # Add child node to stack if exists
            if node.child:
                stack.append(node.child)
            
            # Connect current node
            curr.next = node
            node.prev = curr
            node.child = None
            curr = node
        
        # Fix the head's prev pointer
        dummy.next.prev = None
        return dummy.next

Where n is the total number of nodes in the multilevel linked list.

class Solution {

        if (head == null) {
            return null;
        }

        Node dummy = new Node(0);
        Node curr = dummy;

public class Solution {
    public Node Flatten(Node head) {
        if (head == null) {
            return null;
        }
        
        // Create a dummy node to handle edge cases
        Node dummy = new Node(0);
        Node curr = dummy;
        Stack stack = new Stack();
        stack.Push(head);
        
        while (stack.Count > 0) {
            Node node = stack.Pop();
            
            // Add next node to stack if exists
            if (node.next != null) {
                stack.Push(node.next);
            }
            
            // Add child node to stack if exists
            if (node.child != null) {
                stack.Push(node.child);
            }
            
            // Connect current node
            curr.next = node;
            node.prev = curr;
            node.child = null;
            curr = node;
        }
        
        // Fix the head's prev pointer
        dummy.next.prev = null;
        return dummy.next;
    }
}
