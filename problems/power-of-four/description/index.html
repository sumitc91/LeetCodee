342. Power of Four

Jump to Solution: Python Java C++ JavaScript C#

Problem Description

Given an integer n, return true if it is a power of four. Otherwise, return false.

An integer n is a power of four, if there exists an integer x such that n == 4ˣ.

Example 1:

Input: n = 16
Output: true

Example 2:

Input: n = 5
Output: false

Example 3:

Input: n = 1
Output: true

Constraints:

  • -2³¹ <= n <= 2³¹ - 1

Follow up:

Could you solve it without loops/recursion?

Solution

Python Solution

class Solution:
    def isPowerOfFour(self, n: int) -> bool:
        # Check if n is positive and has only one '1' bit
        # and that '1' bit is at an even position
        return n > 0 and (n & (n - 1)) == 0 and (n & 0xAAAAAAAA) == 0

Time Complexity: O(1)

Constant time bit operations.

Space Complexity: O(1)

No extra space used.
