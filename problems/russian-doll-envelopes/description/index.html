354. Russian Doll Envelopes

Jump to Solution: Python Java C++ JavaScript C#

Problem Description

You are given a 2D array of integers envelopes where envelopes[i] = [wi, hi] represents the width and height of an envelope.

One envelope can fit into another if and only if both the width and height of one envelope are greater than the other envelope's width and height.

Return the maximum number of envelopes you can Russian doll (i.e., put one inside the other).

Note: You cannot rotate an envelope.

Example 1:

Input: envelopes = [[5,4],[6,4],[6,7],[2,3]]
Output: 3
Explanation: The maximum number of envelopes you can Russian doll is 3 ([2,3] => [5,4] => [6,7]).

Example 2:

Input: envelopes = [[1,1],[1,1],[1,1]]
Output: 1

Constraints:

  • 1 <= envelopes.length <= 10⁵
  • envelopes[i].length == 2
  • 1 <= wi, hi <= 10⁵

Solution

Python Solution

class Solution:
    def maxEnvelopes(self, envelopes: List[List[int]]) -> int:
        # Sort by width ascending and height descending
        envelopes.sort(key=lambda x: (x[0], -x[1]))
        
        # Find LIS on heights
        dp = []
        for _, h in envelopes:
            idx = bisect_left(dp, h)
            if idx == len(dp):
                dp.append(h)
            else:
                dp[idx] = h
        
        return len(dp)

Time Complexity: O(n log n)

Where n is the number of envelopes. Sorting takes O(n log n) and finding LIS takes O(n log n).

Space Complexity: O(n)

For storing the dp array.

Java Solution

class Solution {
    public int maxEnvelopes(int[][] envelopes) {
        // Sort by width ascending and height descending
        Arrays.sort(envelopes, (a, b) -> a[0] == b[0] ? b[1] - a[1] : a[0] - b[0]);
        
        // Find LIS on heights
        int[] dp = new int[envelopes.length];
        int len = 0;
        
        for (int[] envelope : envelopes) {
            int idx = Arrays.binarySearch(dp, 0, len, envelope[1]);
            if (idx < 0) {
                idx = -(idx + 1);
            }
            dp[idx] = envelope[1];
            if (idx == len) {
                len++;
            }
        }
        
        return len;
    }
}

C++ Solution

class Solution {
public:
    int maxEnvelopes(vector>& envelopes) {
        // Sort by width ascending and height descending
        sort(envelopes.begin(), envelopes.end(), 
             [](const vector& a, const vector& b) {
                 return a[0] < b[0] || (a[0] == b[0] && a[1] > b[1]);
             });
        
        // Find LIS on heights
        vector dp;
        for (const auto& envelope : envelopes) {
            auto it = lower_bound(dp.begin(), dp.end(), envelope[1]);
            if (it == dp.end()) {
                dp.push_back(envelope[1]);
            } else {
                *it = envelope[1];
            }
        }
        
        return dp.size();
    }
};

JavaScript Solution

/**
 * @param {number[][]} envelopes
 * @return {number}
 */
var maxEnvelopes = function(envelopes) {
    // Sort by width ascending and height descending
    envelopes.sort((a, b) => a[0] === b[0] ? b[1] - a[1] : a[0] - b[0]);
    
    // Find LIS on heights
    const dp = [];
    for (const [_, h] of envelopes) {
        let left = 0, right = dp.length;
        while (left < right) {
            const mid = Math.floor((left + right) / 2);
            if (dp[mid] < h) {
                left = mid + 1;
            } else {
                right = mid;
            }
        }
        dp[left] = h;
    }
    
    return dp.length;
};

C# Solution

public class Solution {
    public int MaxEnvelopes(int[][] envelopes) {
        // Sort by width ascending and height descending
        Array.Sort(envelopes, (a, b) => a[0] == b[0] ? b[1].CompareTo(a[1]) : a[0].CompareTo(b[0]));
        
        // Find LIS on heights
        var dp = new List();
        foreach (var envelope in envelopes) {
            int idx = dp.BinarySearch(envelope[1]);
            if (idx < 0) {
                idx = ~idx;
            }
            if (idx == dp.Count) {
                dp.Add(envelope[1]);
            } else {
                dp[idx] = envelope[1];
            }
        }
        
        return dp.Count;
    }
}

Approach Explanation

The solution uses a combination of sorting and Longest Increasing Subsequence (LIS):

  1. Key Insight:
    • Sort by width
    • Height LIS
    • Binary search
    • Optimal substructure
  2. Algorithm Steps:
    • Sort envelopes
    • Process heights
    • Find LIS
    • Return length

Sorting Strategy:

  • Width ascending
  • Height descending
  • Prevents duplicates
  • Ensures nesting

Optimization insights:

  • Binary search
  • Smart sorting
  • Space efficient
  • No backtracking

Edge Cases:

  • Single envelope
  • Same dimensions
