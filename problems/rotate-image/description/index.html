48. Rotate Image

Medium

Problem Description

You are given an n x n 2D matrix representing an image, rotate the image by 90 degrees (clockwise).

You have to rotate the image in-place, which means you have to modify the input 2D matrix directly. DO NOT allocate another 2D matrix and do the rotation.

Examples

Example 1:
Input: matrix = [[1,2,3],[4,5,6],[7,8,9]]
Output: [[7,4,1],[8,5,2],[9,6,3]]

Example 2:
Input: matrix = [[5,1,9,11],[2,4,8,10],[13,3,6,7],[15,14,12,16]]
Output: [[15,13,2,5],[14,3,4,1],[12,6,8,9],[16,7,10,11]]

Python Solution


def rotate(matrix: List[List[int]]) -> None:
    n = len(matrix)
    
    # Step 1: Transpose matrix
    for i in range(n):
        for j in range(i, n):
            matrix[i][j], matrix[j][i] = matrix[j][i], matrix[i][j]
    
    # Step 2: Reverse each row
    for i in range(n):
        matrix[i].reverse()

Java Solution


class Solution {
    public void rotate(int[][] matrix) {
        int n = matrix.length;
        
        // Step 1: Transpose matrix
        for (int i = 0; i < n; i++) {
            for (int j = i; j < n; j++) {
                int temp = matrix[i][j];
                matrix[i][j] = matrix[j][i];
                matrix[j][i] = temp;
            }
        }
        
        // Step 2: Reverse each row
        for (int i = 0; i < n; i++) {
            for (int j = 0; j < n/2; j++) {
                int temp = matrix[i][j];
                matrix[i][j] = matrix[i][n-1-j];
                matrix[i][n-1-j] = temp;
            }
        }
    }
}

C++ Solution


class Solution {
public:
    void rotate(vector>& matrix) {
        int n = matrix.size();
        
        // Step 1: Transpose matrix
        for (int i = 0; i < n; i++) {
            for (int j = i; j < n; j++) {
                swap(matrix[i][j], matrix[j][i]);
            }
        }
        
        // Step 2: Reverse each row
        for (int i = 0; i < n; i++) {
            reverse(matrix[i].begin(), matrix[i].end());
        }
    }
};

    // Step 1: Transpose matrix
    for (let i = 0; i < n; i++) {

    for (let i = 0; i < n; i++) {
        matrix[i].reverse();
    }
};

        int n = matrix.Length;
        
        // Step 1: Transpose matrix
        for (int i = 0; i < n; i++) {
            for (int j = i; j < n; j++) {

Solution Explanation

This solution rotates the matrix in two steps to achieve in-place rotation. Here's how it works:

  • Step 1: Transpose matrix
    • Swap elements across main diagonal
    • matrix[i][j] becomes matrix[j][i]
  • Step 2: Reverse each row
    • Reverse elements in each row
    • First element becomes last

Key points:

  • In-place rotation without extra space
  • Two-step process: transpose then reverse
  • Works for any square matrix
  • Maintains original values
