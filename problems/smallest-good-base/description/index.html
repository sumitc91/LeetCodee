483. Smallest Good Base

Jump to Solution: Python Java C++ JavaScript C#

Problem Description

Given a string n representing an integer, return the smallest good base of n.

We call k >= 2 a good base of n, if all digits of n base k are 1's.

Example 1:

Input: n = "13"
Output: "3"
Explanation: 13 base 3 is 111.

Example 2:

Input: n = "4681"
Output: "8"
Explanation: 4681 base 8 is 11111.

Example 3:

Input: n = "1000000000000000000"
Output: "999999999999999999"
Explanation: 1000000000000000000 base 999999999999999999 is 11.

Constraints:

  • n is an integer in the range [3, 10¹⁸].
  • n does not contain any leading zeros.

Solution

Python Solution

class Solution:
    def smallestGoodBase(self, n: str) -> str:
        n = int(n)
        max_m = len(bin(n)) - 2  # Maximum possible length in any base
        
        # Try each possible length m
        for m in range(max_m, 1, -1):
            # For length m, solve (k^m - 1) = n*(k - 1)
            # Use binary search to find k
            left = 2
            right = n - 1
            
            while left <= right:
                k = (left + right) // 2
                value = (pow(k, m) - 1) // (k - 1)
                
                if value == n:
                    return str(k)
                elif value < n:
                    left = k + 1
                else:
                    right = k - 1
        
        return str(n - 1)

Time Complexity: O(log n × log n)

Where n is the input number. We try O(log n) possible lengths and for each length, binary search takes O(log n) time.

Space Complexity: O(1)

Only constant extra space is used.

Java Solution

class Solution {
    public String smallestGoodBase(String n) {
        long num = Long.parseLong(n);
        int maxM = (int)(Math.log(num) / Math.log(2));  // Maximum possible length in any base
        
        // Try each possible length m
        for (int m = maxM; m > 1; m--) {
            // For length m, solve (k^m - 1) = n*(k - 1)
            // Use binary search to find k
            long left = 2;
            long right = num - 1;
            
            while (left <= right) {
                long k = left + (right - left) / 2;
                long value = 0;
                long curr = 1;
                boolean overflow = false;
                
                // Calculate k^m - 1
                for (int i = 0; i <= m && !overflow; i++) {
                    value += curr;
                    if (i < m) {
                        if (curr > num / k) {
                            overflow = true;
                        }
                        curr *= k;
                    }
                }
                
                if (!overflow && value == num) {
                    return String.valueOf(k);
                }
                
                if (overflow || value > num) {
                    right = k - 1;
                } else {
                    left = k + 1;
                }
            }
        }
        
        return String.valueOf(num - 1);
    }
}

var smallestGoodBase = function(n) {
    n = BigInt(n);
    const maxM = Math.floor(Math.log2(Number(n)));  // Maximum possible length in any base
    
    // Try each possible length m
    for (let m = maxM; m > 1; m--) {
        // For length m, solve (k^m - 1) = n*(k - 1)
        // Use binary search to find k
        let left = 2n;
        let right = n - 1n;
        
        while (left <= right) {
            const k = left + (right - left) / 2n;
            let value = 0n;
            let curr = 1n;
            let overflow = false;
            
            // Calculate k^m - 1
            for (let i = 0; i <= m && !overflow; i++) {
                value += curr;
                if (i < m) {
                    if (curr > n / k) {
                        overflow = true;
                    }
                    curr *= k;
                }
            }
            
            if (!overflow && value === n) {
                return k.toString();
            }
            
            if (overflow || value > n) {
                right = k - 1n;
            } else {
                left = k + 1n;
            }
        }
    }
    
    return (n - 1n).toString();
};
