324. Wiggle Sort II

Jump to Solution: Python Java C++ JavaScript C#

Problem Description

Given an integer array nums, reorder it such that nums[0] < nums[1] > nums[2] < nums[3]...

You may assume the input array always has a valid answer.

Example 1:

Input: nums = [1,5,1,1,6,4]
Output: [1,6,1,5,1,4]

Example 2:

Input: nums = [1,3,2,2,3,1]
Output: [2,3,1,3,1,2]

Constraints:

  • 1 <= nums.length <= 5 * 10⁴
  • 0 <= nums[i] <= 5000
  • It is guaranteed that there will be an answer for the given input nums.

Follow Up:

Can you do it in O(n) time and/or in-place with O(1) extra space?

Solution

Python Solution

class Solution:
    def wiggleSort(self, nums: List[int]) -> None:
        """
        Do not return anything, modify nums in-place instead.
        """
        # Step 1: Sort the array
        nums.sort()
        n = len(nums)
        
        # Step 2: Create a new array with wiggle sort
        result = [0] * n
        mid = (n - 1) // 2
        end = n - 1
        
        # Step 3: Fill odd indices with larger numbers
        for i in range(1, n, 2):
            if i < n:
                result[i] = nums[end]
                end -= 1
        
        # Step 4: Fill even indices with smaller numbers
        for i in range(0, n, 2):
            if i < n:
                result[i] = nums[mid]
                mid -= 1
        
        # Step 5: Copy back to original array
        for i in range(n):
            nums[i] = result[i]

Time Complexity: O(n log n)

Due to sorting the array.

Space Complexity: O(n)

For the temporary array.

Java Solution

class Solution {
    public void wiggleSort(int[] nums) {
        // Step 1: Sort the array
        Arrays.sort(nums);
        int n = nums.length;
        
        // Step 2: Create a new array with wiggle sort
        int[] result = new int[n];
        int mid = (n - 1) / 2;
        int end = n - 1;
        
        // Step 3: Fill odd indices with larger numbers
        for (int i = 1; i < n; i += 2) {
            result[i] = nums[end];
            end--;
        }
        
        // Step 4: Fill even indices with smaller numbers
        for (int i = 0; i < n; i += 2) {
            result[i] = nums[mid];
            mid--;
        }
        
        // Step 5: Copy back to original array
        System.arraycopy(result, 0, nums, 0, n);
    }
}
