410. Split Array Largest Sum

Jump to Solution: Python Java C++ JavaScript C#

Problem Description

Given an array nums which consists of non-negative integers and an integer k, split the array into k non-empty continuous subarrays.

The largest sum among these k subarrays is the objective function to minimize.

Return the minimized largest sum of the split.

Example 1:

Input: nums = [7,2,5,10,8], k = 2
Output: 18
Explanation: There are four ways to split nums into two subarrays.
The best way is to split it into [7,2,5] and [10,8], where the largest sum among the two subarrays is only 18.

Example 2:

Input: nums = [1,2,3,4,5], k = 2
Output: 9
Explanation: There are four ways to split nums into two subarrays.
The best way is to split it into [1,2,3] and [4,5], where the largest sum among the two subarrays is only 9.

Constraints:

  • 1 <= nums.length <= 1000
  • 0 <= nums[i] <= 10⁶
  • 1 <= k <= min(50, nums.length)

Solution

Python Solution

class Solution:
    def splitArray(self, nums: List[int], k: int) -> int:
        def canSplit(largest_sum: int) -> bool:
            count = 1
            current_sum = 0
            
            for num in nums:
                if num > largest_sum:
                    return False
                if current_sum + num > largest_sum:
                    count += 1
                    current_sum = num
                else:
                    current_sum += num
            
            return count <= k
        
        left = max(nums)
        right = sum(nums)
        
        while left < right:
            mid = left + (right - left) // 2
            if canSplit(mid):
                right = mid
            else:
                left = mid + 1
        
        return left

Time Complexity: O(n * log(sum))

Where n is the length of nums and sum is the sum of all elements. Binary search takes O(log(sum)) iterations and each iteration needs O(n) to check if it's possible.

Space Complexity: O(1)

Only constant extra space is used.

class Solution {
    public int splitArray(int[] nums, int k) {
        long left = 0;
        long right = 0;
        
        // Find the range for binary search
        for (int num : nums) {
            left = Math.max(left, num);
            right += num;
        }
        
        while (left < right) {
            long mid = left + (right - left) / 2;
            if (canSplit(nums, k, mid)) {
                right = mid;
            } else {
                left = mid + 1;
            }

var splitArray = function(nums, k) {
    const canSplit = (largestSum) => {
        let count = 1;
        let currentSum = 0;
        
        for (const num of nums) {
            if (currentSum + num > largestSum) {
                count++;
                currentSum = num;
            } else {
                currentSum += num;
            }
        }
        
        return count <= k;
    };
    
    let left = Math.max(...nums);
    let right = nums.reduce((a, b) => a + b, 0);

        const mid = Math.floor(left + (right - left) / 2);
        if (canSplit(mid)) {
            right = mid;
        } else {
            left = mid + 1;
        }
    }
    
    return left;
};

    public int SplitArray(int[] nums, int k) {
        long left = 0;
        long right = 0;
        
        // Find the range for binary search
        foreach (int num in nums) {
            left = Math.Max(left, num);
            right += num;
        }
        
        while (left < right) {
            long mid = left + (right - left) / 2;
            if (CanSplit(nums, k, mid)) {
                right = mid;
            } else {
                left = mid + 1;
            }
        }
        
        return (int)left;
    }
    
    private bool CanSplit(int[] nums, int k, long largestSum) {
        int count = 1;

                currentSum += num;
            }
        }
        
        return count <= k;
    }
}
