507. Perfect Number

Jump to Solution: Python Java C++ JavaScript C#

Problem Description

A perfect number is a positive integer that is equal to the sum of its positive divisors, excluding the number itself. A divisor of an integer x is an integer that can divide x evenly.

Given an integer n, return true if n is a perfect number, otherwise return false.

Example 1:

Input: num = 28
Output: true
Explanation: 28 = 1 + 2 + 4 + 7 + 14
1, 2, 4, 7, and 14 are all divisors of 28.

Example 2:

Input: num = 7
Output: false

Constraints:

  • 1 <= num <= 10⁸

Solution

Python Solution

class Solution:
    def checkPerfectNumber(self, num: int) -> bool:
        if num <= 1:
            return False
        
        divisor_sum = 1  # 1 is always a divisor
        sqrt_num = int(num ** 0.5)
        
        # Check divisors up to square root
        for i in range(2, sqrt_num + 1):
            if num % i == 0:
                divisor_sum += i
                # Add the pair divisor if it's different
                if i != num // i:
                    divisor_sum += num // i
        
        return divisor_sum == num

Time Complexity: O(√n)

Where n is the input number. We only need to check divisors up to the square root.

Space Complexity: O(1)

We only use a constant amount of extra space.

/**
 * @param {number} num
 * @return {boolean}
 */
var checkPerfectNumber = function(num) {
    if (num <= 1) {
        return false;
    }
    
    let divisorSum = 1;  // 1 is always a divisor
    const sqrtNum = Math.floor(Math.sqrt(num));
    
    // Check divisors up to square root
    for (let i = 2; i <= sqrtNum; i++) {
        if (num % i === 0) {
            divisorSum += i;
            // Add the pair divisor if it's different
            if (i !== num / i) {
                divisorSum += num / i;
            }
        }
    }
    
    return divisorSum === num;
};
