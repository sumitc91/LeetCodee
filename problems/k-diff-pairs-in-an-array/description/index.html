532. K-diff Pairs in an Array

Jump to Solution: Python Java C++ JavaScript C#

Problem Description

Given an array of integers nums and an integer k, return the number of unique k-diff pairs in the array.

A k-diff pair is an integer pair (nums[i], nums[j]), where the following are true:

  • 0 <= i, j < nums.length
  • i != j
  • nums[i] - nums[j] == k

Example 1:

Input: nums = [3,1,4,1,5], k = 2
Output: 2
Explanation: There are two 2-diff pairs in the array, (1, 3) and (3, 5).
Although we have two 1s in the input, we should only return the number of unique pairs.

Example 2:

Input: nums = [1,2,3,4,5], k = 1
Output: 4
Explanation: There are four 1-diff pairs in the array, (1, 2), (2, 3), (3, 4) and (4, 5).

Example 3:

Input: nums = [1,3,1,5,4], k = 0
Output: 1
Explanation: There is one 0-diff pair in the array, (1, 1).

Constraints:

  • 1 <= nums.length <= 10⁴
  • -10⁷ <= nums[i] <= 10⁷
  • 0 <= k <= 10⁷

Solution

Python Solution

class Solution:
    def findPairs(self, nums: List[int], k: int) -> int:
        count = Counter(nums)
        result = 0
        
        for num in count:
            if k > 0 and num + k in count:
                result += 1
            elif k == 0 and count[num] > 1:
                result += 1
                
        return result

Time Complexity: O(n)

Where n is the length of nums. We iterate through the array once to build the counter and once through unique numbers.

Space Complexity: O(n)

For storing the counter dictionary.

Java Solution

class Solution {
    public int findPairs(int[] nums, int k) {
        Map count = new HashMap<>();
        for (int num : nums) {
            count.put(num, count.getOrDefault(num, 0) + 1);
        }
        
        int result = 0;
        for (int num : count.keySet()) {
            if (k > 0 && count.containsKey(num + k)) {
                result++;
            } else if (k == 0 && count.get(num) > 1) {
                result++;
            }
        }
        
        return result;
    }
}

class Solution {
public:
    int findPairs(vector& nums, int k) {
        unordered_map count;
        for (int num : nums) {
            count[num]++;
        }
        
        int result = 0;
        for (const auto& pair : count) {

                result++;
            } else if (k == 0 && pair.second > 1) {
                result++;
            }
        }
        
        return result;
    }

var findPairs = function(nums, k) {
    const count = new Map();
    for (const num of nums) {
        count.set(num, (count.get(num) || 0) + 1);
    }
    
    let result = 0;
    for (const [num, freq] of count) {
        if (k > 0 && count.has(num + k)) {
            result++;
        } else if (k === 0 && freq > 1) {
            result++;
        }
    }
    
    return result;
};

C# Solution

public class Solution {
    public int FindPairs(int[] nums, int k) {
        Dictionary count = new Dictionary();
        foreach (int num in nums) {
            if (!count.ContainsKey(num)) {
                count[num] = 0;
            }
            count[num]++;
        }
        
        int result = 0;
        foreach (var pair in count) {
            if (k > 0 && count.ContainsKey(pair.Key + k)) {
                result++;
            } else if (k == 0 && pair.Value > 1) {
                result++;
            }
        }
        
        return result;
    }
}

Time Complexity: O(n)

Where n is the length of nums. We iterate through the array once to build the dictionary and once through unique numbers.

Space Complexity: O(n)

For storing the dictionary.
