542. 01 Matrix

Jump to Solution: Python Java C++ JavaScript C#

Problem Description

Given an m x n binary matrix mat, return the distance of the nearest 0 for each cell.

The distance between two adjacent cells is 1.

Example 1:

Input: mat = [[0,0,0],[0,1,0],[0,0,0]]
Output: [[0,0,0],[0,1,0],[0,0,0]]
Explanation: All zeros are at distance 0 from themselves, and the 1 at position (1,1) has nearest 0 at distance 1.

Example 2:

Input: mat = [[0,0,0],[0,1,0],[1,1,1]]
Output: [[0,0,0],[0,1,0],[1,2,1]]
Explanation: The zeros are at distance 0, the 1s at position (2,0) and (2,2) are at distance 1, and the 1 at position (2,1) is at distance 2.

Constraints:

  • m == mat.length
  • n == mat[i].length
  • 1 <= m, n <= 10⁴
  • 1 <= m * n <= 10⁴
  • mat[i][j] is either 0 or 1
  • There is at least one 0 in mat

Solution

Python Solution

class Solution:
    def updateMatrix(self, mat: List[List[int]]) -> List[List[int]]:
        m, n = len(mat), len(mat[0])
        dist = [[float('inf')] * n for _ in range(m)]
        queue = deque()
        
        # Initialize queue with all 0s and mark their distances as 0
        for i in range(m):
            for j in range(n):
                if mat[i][j] == 0:
                    dist[i][j] = 0
                    queue.append((i, j))
        
        # BFS to update distances
        directions = [(1,0), (-1,0), (0,1), (0,-1)]
        while queue:
            i, j = queue.popleft()
            for di, dj in directions:
                ni, nj = i + di, j + dj
                if 0 <= ni < m and 0 <= nj < n and dist[ni][nj] > dist[i][j] + 1:
                    dist[ni][nj] = dist[i][j] + 1
                    queue.append((ni, nj))
        
        return dist

Time Complexity: O(m * n)

Where m and n are the dimensions of the matrix. We visit each cell at most once.

Space Complexity: O(m * n)

For the distance matrix and queue.

Java Solution

class Solution {
    public int[][] updateMatrix(int[][] mat) {
        int m = mat.length, n = mat[0].length;
        int[][] dist = new int[m][n];
        Queue queue = new LinkedList<>();
        
        // Initialize distances and queue
        for (int i = 0; i < m; i++) {
            for (int j = 0; j < n; j++) {
                if (mat[i][j] == 0) {
                    dist[i][j] = 0;
                    queue.offer(new int[]{i, j});
                } else {
                    dist[i][j] = Integer.MAX_VALUE;
                }
            }
        }
        
        // BFS to update distances
        int[][] directions = {{1,0}, {-1,0}, {0,1}, {0,-1}};
        while (!queue.isEmpty()) {
            int[] curr = queue.poll();
            for (int[] dir : directions) {
                int ni = curr[0] + dir[0];
                int nj = curr[1] + dir[1];
                if (ni >= 0 && ni < m && nj >= 0 && nj < n && 
                    dist[ni][nj] > dist[curr[0]][curr[1]] + 1) {
                    dist[ni][nj] = dist[curr[0]][curr[1]] + 1;
                    queue.offer(new int[]{ni, nj});
                }
            }
        }
        
        return dist;
    }
}

C# Solution

public class Solution {
    public int[][] UpdateMatrix(int[][] mat) {
        int m = mat.Length, n = mat[0].Length;
        int[][] dist = new int[m][];
        Queue<(int, int)> queue = new Queue<(int, int)>();
        
        // Initialize distances and queue
        for (int i = 0; i < m; i++) {
            dist[i] = new int[n];
            for (int j = 0; j < n; j++) {
                if (mat[i][j] == 0) {
                    dist[i][j] = 0;
                    queue.Enqueue((i, j));
                } else {
                    dist[i][j] = int.MaxValue;
                }
            }
        }
        
        // BFS to update distances
        (int, int)[] directions = {(1,0), (-1,0), (0,1), (0,-1)};
        while (queue.Count > 0) {
            var (i, j) = queue.Dequeue();
            foreach (var (di, dj) in directions) {
                int ni = i + di, nj = j + dj;
                if (ni >= 0 && ni < m && nj >= 0 && nj < n && 
                    dist[ni][nj] > dist[i][j] + 1) {
                    dist[ni][nj] = dist[i][j] + 1;
                    queue.Enqueue((ni, nj));
                }
            }
        }
        
        return dist;
    }
}
