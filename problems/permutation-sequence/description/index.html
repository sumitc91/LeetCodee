60. Permutation Sequence

Hard

Problem Description

The set [1, 2, 3, ..., n] contains a total of n! unique permutations.

Given n and k, return the kth permutation sequence.

Examples

Example 1:
Input: n = 3, k = 3
Output: "213"

Example 2:
Input: n = 4, k = 9
Output: "2314"

Example 3:
Input: n = 3, k = 1
Output: "123"

Python Solution


def getPermutation(n: int, k: int) -> str:
    numbers = list(range(1, n + 1))
    result = []
    k = k - 1  # Convert to 0-based index
    
    # Calculate factorials
    factorial = [1]
    for i in range(1, n):
        factorial.append(factorial[-1] * i)
    
    # Build the permutation
    for i in range(n - 1, -1, -1):
        index = k // factorial[i]
        k = k % factorial[i]
        
        result.append(str(numbers[index]))
        numbers.pop(index)
    
    return ''.join(result)

Java Solution


class Solution {
    public String getPermutation(int n, int k) {
        List numbers = new ArrayList<>();
        int[] factorial = new int[n];
        StringBuilder result = new StringBuilder();
        
        // Initialize numbers and factorial array
        factorial[0] = 1;
        for (int i = 1; i < n; i++) {
            factorial[i] = factorial[i-1] * i;
        }
        for (int i = 1; i <= n; i++) {
            numbers.add(i);
        }
        
        k = k - 1; // Convert to 0-based index
        
        // Build the permutation
        for (int i = n - 1; i >= 0; i--) {
            int index = k / factorial[i];
            k = k % factorial[i];
            
            result.append(numbers.get(index));
            numbers.remove(index);
        }
        
        return result.toString();
    }
}

 * @param {number} k
 * @return {string}

var getPermutation = function(n, k) {
    const numbers = [];
    const factorial = [1];
    let result = '';
    
    // Initialize numbers and factorial array
    for (let i = 1; i < n; i++) {
        factorial[i] = factorial[i-1] * i;
    }
    for (let i = 1; i <= n; i++) {
        numbers.push(i);
    }
    
    k = k - 1; // Convert to 0-based index
    
    // Build the permutation
    for (let i = n - 1; i >= 0; i--) {
        const index = Math.floor(k / factorial[i]);
        k = k % factorial[i];
        
        result += numbers[index];
        numbers.splice(index, 1);
    }
    
    return result;
};

public class Solution {
    public string GetPermutation(int n, int k) {
        List numbers = new List();
        int[] factorial = new int[n];
        StringBuilder result = new StringBuilder();
        
        // Initialize numbers and factorial array
        factorial[0] = 1;
        for (int i = 1; i < n; i++) {
            factorial[i] = factorial[i-1] * i;
        }
        for (int i = 1; i <= n; i++) {
            numbers.Add(i);
        }
        
        k = k - 1; // Convert to 0-based index
        
        // Build the permutation
        for (int i = n - 1; i >= 0; i--) {
            int index = k / factorial[i];
            k = k % factorial[i];
            
            result.Append(numbers[index]);
            numbers.RemoveAt(index);
        }
        
        return result.ToString();
    }
}

Complexity Analysis

  • Time Complexity: O(n) where n is the input number
  • Space Complexity: O(n) to store the numbers and factorials

Solution Explanation

This solution uses a mathematical approach to find the kth permutation:

  • Key concepts:
    • Use factorial numbers to determine position
    • Calculate index for each position
    • Remove used numbers from available set
  • Algorithm steps:
    • Calculate factorials up to n-1
    • Convert k to 0-based index
    • For each position:
      • Calculate index using division
      • Update k using modulo
      • Add number at index to result
      • Remove used number

Key points:

  • No need to generate all permutations
  • Efficient mathematical solution
  • Handles all valid inputs
  • Linear time complexity
