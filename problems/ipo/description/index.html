502. IPO

Jump to Solution: Python Java C++ JavaScript C#

Problem Description

Suppose LeetCode will start its IPO soon. In order to sell a good price of its shares to Venture Capital, LeetCode would like to work on some projects to increase its capital before the IPO. Since it has limited resources, it can only finish at most k distinct projects before the IPO. Help LeetCode design the best way to maximize its total capital after finishing at most k distinct projects.

You are given n projects where the ith project has a pure profit profits[i] and a minimum capital of capital[i] is needed to start it.

Initially, you have w capital. When you finish a project, you will obtain its pure profit and the profit will be added to your total capital.

Pick a list of at most k distinct projects from given projects to maximize your final capital, and return the final maximized capital.

The answer is guaranteed to fit in a 32-bit signed integer.

Example 1:

Input: k = 2, w = 0, profits = [1,2,3], capital = [0,1,1]
Output: 4
Explanation: Since your initial capital is 0, you can only start the project indexed 0.
After finishing it you will obtain profit 1 and your capital becomes 1.
With capital 1, you can either start the project indexed 1 or the project indexed 2.
Since you can choose at most 2 projects, you need to finish the project indexed 2 to get the maximum capital.
Therefore, output the final maximized capital, which is 0 + 1 + 3 = 4.

Example 2:

Input: k = 3, w = 0, profits = [1,2,3], capital = [0,1,2]
Output: 6

Constraints:

  • 1 <= k <= 10⁵
  • 0 <= w <= 10⁹
  • n == profits.length
  • n == capital.length
  • 1 <= n <= 10⁵
  • 0 <= profits[i] <= 10⁴
  • 0 <= capital[i] <= 10⁹

Solution

Python Solution

class Solution:
    def findMaximizedCapital(self, k: int, w: int, profits: List[int], capital: List[int]) -> int:
        n = len(profits)
        projects = list(zip(capital, profits))
        projects.sort()  # Sort by capital
        
        i = 0
        available = []  # max heap of profits
        
        # Do k projects
        for _ in range(k):
            # Add all projects we can afford with current capital
            while i < n and projects[i][0] <= w:
                heapq.heappush(available, -projects[i][1])
                i += 1
            
            if not available:
                break
                
            # Take the most profitable project
            w -= heapq.heappop(available)
        
        return w

Java Solution

class Solution {
    public int findMaximizedCapital(int k, int w, int[] profits, int[] capital) {
        int n = profits.length;
        int[][] projects = new int[n][2];
        
        // Combine profits and capital
        for (int i = 0; i < n; i++) {
            projects[i][0] = capital[i];
            projects[i][1] = profits[i];
        }
        
        // Sort by capital
        Arrays.sort(projects, (a, b) -> a[0] - b[0]);
        
        // Max heap for profits
        PriorityQueue available = new PriorityQueue<>((a, b) -> b - a);
        
        int i = 0;
        // Do k projects
        while (k > 0) {
            // Add all projects we can afford
            while (i < n && projects[i][0] <= w) {
                available.offer(projects[i][1]);
                i++;
            }
            
            if (available.isEmpty()) {
                break;
            }
            
            // Take the most profitable project
            w += available.poll();
            k--;
        }
        
        return w;
    }
}

C# Solution

public class Solution {
    public int FindMaximizedCapital(int k, int w, int[] profits, int[] capital) {
        int n = profits.Length;
        var projects = new List<(int capital, int profit)>();
        
        // Combine profits and capital
        for (int i = 0; i < n; i++) {
            projects.Add((capital[i], profits[i]));
        }
        
        // Sort by capital
        projects.Sort((a, b) => a.capital.CompareTo(b.capital));
        
        // Max heap for profits
        var available = new PriorityQueue();
        
        int i = 0;
        // Do k projects
        while (k > 0) {
            // Add all projects we can afford
            while (i < n && projects[i].capital <= w) {
                available.Enqueue(projects[i].profit, -projects[i].profit);
                i++;
            }
            
            if (available.Count == 0) {
                break;
            }
            
            // Take the most profitable project
            w += available.Dequeue();
            k--;
        }
        
        return w;
    }
}
