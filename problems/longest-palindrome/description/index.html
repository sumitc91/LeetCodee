409. Longest Palindrome

Jump to Solution: Python Java C++ JavaScript C#

Problem Description

Given a string s which consists of lowercase or uppercase letters, return the length of the longest palindrome that can be built with those letters.

Letters are case sensitive, for example, "Aa" is not considered a palindrome here.

Example 1:

Input: s = "abccccdd"
Output: 7
Explanation: One longest palindrome that can be built is "dccaccd", whose length is 7.

Example 2:

Input: s = "a"
Output: 1

Example 3:

Input: s = "bb"
Output: 2

Constraints:

  • 1 <= s.length <= 2000
  • s consists of lowercase and/or uppercase English letters only.

Solution

Python Solution

class Solution:
    def longestPalindrome(self, s: str) -> int:
        # Count frequency of each character
        char_count = Counter(s)
        
        # Initialize result and odd count flag
        result = 0
        has_odd = False
        
        # Process each character count
        for count in char_count.values():
            # Add even counts directly
            result += (count // 2) * 2
            # Check for odd counts
            if count % 2 == 1:
                has_odd = True
        
        # Add one more character if we have any odd counts
        return result + (1 if has_odd else 0)

Time Complexity: O(n)

Where n is the length of the string. We need to count each character once.

Space Complexity: O(1)

Since we only store counts for a fixed set of characters (52 for uppercase and lowercase letters).

Java Solution

class Solution {
    public int longestPalindrome(String s) {
        // Count frequency of each character
        int[] charCount = new int[128];
        for (char c : s.toCharArray()) {
            charCount[c]++;
        }
        
        // Initialize result and odd count flag
        int result = 0;
        boolean hasOdd = false;
        
        // Process each character count
        for (int count : charCount) {
            // Add even counts directly
            result += (count / 2) * 2;
            // Check for odd counts
            if (count % 2 == 1) {
                hasOdd = true;
            }
        }
        
        // Add one more character if we have any odd counts
        return result + (hasOdd ? 1 : 0);
    }
}

var longestPalindrome = function(s) {
    // Count frequency of each character
    const charCount = new Map();
    for (const c of s) {
        charCount.set(c, (charCount.get(c) || 0) + 1);
    }

    let hasOdd = false;

            hasOdd = true;
        }
