326. Power of Three

Jump to Solution: Python Java C++ JavaScript C#

Problem Description

Given an integer n, return true if it is a power of three. Otherwise, return false.

An integer n is a power of three, if there exists an integer x such that n == 3ˣ.

Example 1:

Input: n = 27
Output: true
Explanation: 27 = 3³

Example 2:

Input: n = 0
Output: false
Explanation: There is no x where 3ˣ = 0.

Example 3:

Input: n = -1
Output: false
Explanation: There is no x where 3ˣ = (-1).

Constraints:

  • -2³¹ <= n <= 2³¹ - 1

Follow up:

Could you solve it without loops/recursion?

Solution

Python Solution

class Solution:
    def isPowerOfThree(self, n: int) -> bool:
        # Method 1: Using logarithm
        if n <= 0:
            return False
        # log₃(n) = log(n)/log(3) should be an integer
        x = log(n) / log(3)
        return abs(x - round(x)) < 1e-10
        
        # Method 2: Using maximum power of 3 in integer range
        # return n > 0 and 3**19 % n == 0

Time Complexity: O(1)

Both methods use constant time operations.

Space Complexity: O(1)

Only using constant extra space.

Approach Explanation

The solution provides two different approaches:

  1. Logarithmic Method:
    • Use mathematical properties
    • Handle floating-point precision
    • Check integer property
    • Handle edge cases
  2. Maximum Power Method:
    • Use integer constraints
    • Find maximum power
    • Use modulo operation
    • Handle divisibility

Mathematical insight:

  • log₃(n) is integer for powers of 3
  • 3¹⁹ is largest power in 32-bit
  • Divisibility property
  • Floating-point precision

Optimization insights:

  • No loops needed
  • Constant time solution
  • Handle precision errors
  • Integer arithmetic

Edge Cases:

  • Negative numbers
  • Zero input
