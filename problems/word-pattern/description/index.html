290. Word Pattern

Jump to Solution: Python Java C++ JavaScript C#

Problem Description

Given a pattern and a string s, find if s follows the same pattern.

Here follow means a full match, such that there is a bijection between a letter in pattern and a non-empty word in s.

Example 1:

Input: pattern = "abba", s = "dog cat cat dog"
Output: true

Example 2:

Input: pattern = "abba", s = "dog cat cat fish"
Output: false

Example 3:

Input: pattern = "aaaa", s = "dog cat cat dog"
Output: false

Constraints:

  • 1 <= pattern.length <= 300
  • pattern contains only lower-case English letters.
  • 1 <= s.length <= 3000
  • s contains only lowercase English letters and spaces ' '.
  • s does not contain any leading or trailing spaces.
  • All the words in s are separated by a single space.

Solution

Python Solution

class Solution:
    def wordPattern(self, pattern: str, s: str) -> bool:
        words = s.split()
        
        # Check if lengths match
        if len(pattern) != len(words):
            return False
            
        # Create mappings
        char_to_word = {}
        word_to_char = {}
        
        # Check pattern matching
        for c, w in zip(pattern, words):
            if c not in char_to_word and w not in word_to_char:
                char_to_word[c] = w
                word_to_char[w] = c
            elif char_to_word.get(c) != w or word_to_char.get(w) != c:
                return False
                
        return True

Time Complexity: O(n)

Where n is the length of the pattern.

Space Complexity: O(n)

For storing the mappings.

Java Solution

class Solution {
    public boolean wordPattern(String pattern, String s) {
        String[] words = s.split(" ");
        
        // Check if lengths match
        if (pattern.length() != words.length) {
            return false;
        }
        
        // Create mappings
        Map charToWord = new HashMap<>();
        Map wordToChar = new HashMap<>();
        
        // Check pattern matching
        for (int i = 0; i < pattern.length(); i++) {
            char c = pattern.charAt(i);
            String w = words[i];
            
            if (!charToWord.containsKey(c) && !wordToChar.containsKey(w)) {
                charToWord.put(c, w);
                wordToChar.put(w, c);
            } else if (!w.equals(charToWord.get(c)) || c != wordToChar.get(w)) {
                return false;
            }
        }
        
        return true;
    }
}

class Solution {

    bool wordPattern(string pattern, string s) {
        // Split string into words
        vector words;
        stringstream ss(s);
        string word;
        while (ss >> word) {
            words.push_back(word);
        }
        
        // Check if lengths match
        if (pattern.length() != words.size()) {
            return false;
        }
        
        // Create mappings
        unordered_map charToWord;
        unordered_map wordToChar;
        
        // Check pattern matching

            char c = pattern[i];
            string w = words[i];

                wordToChar.find(w) == wordToChar.end()) {
                charToWord[c] = w;
                wordToChar[w] = c;
            } else if (charToWord[c] != w || wordToChar[w] != c) {
                return false;
            }
        }
        
        return true;
    }
};

public class Solution {
    public bool WordPattern(string pattern, string s) {
        string[] words = s.Split(' ');
        
        // Check if lengths match
        if (pattern.Length != words.Length) {
            return false;
        }
        
        // Create mappings
        Dictionary charToWord = new Dictionary();
        Dictionary wordToChar = new Dictionary();
        
        // Check pattern matching
        for (int i = 0; i < pattern.Length; i++) {
            char c = pattern[i];
            string w = words[i];
            
            if (!charToWord.ContainsKey(c) && !wordToChar.ContainsKey(w)) {
                charToWord[c] = w;
                wordToChar[w] = c;
            } else if (!w.Equals(charToWord.GetValueOrDefault(c)) || 
                       c != wordToChar.GetValueOrDefault(w)) {
                return false;
            }
        }
        
        return true;
    }
}

Approach Explanation

The solution uses two hash maps to maintain bijection:

  1. Key Insight:
    • Need two-way mapping
    • Pattern length must match words count
    • Each character maps to unique word
    • Each word maps to unique character
  2. Algorithm Steps:
    • Split string into words
    • Check length equality
    • Build mappings simultaneously
    • Verify consistency

Example walkthrough for "abba" and "dog cat cat dog":

  • a -> dog, dog -> a
  • b -> cat, cat -> b
  • b -> cat (matches)
  • a -> dog (matches)

Optimization insights:

  • Early length check
  • Single pass solution
  • Efficient string splitting
  • Hash map for O(1) lookups

Edge Cases:

  • Empty pattern/string
  • Single character/word
  • All same characters/words
  • No matching pattern
