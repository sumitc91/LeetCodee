279. Perfect Squares

Jump to Solution: Python Java C++ JavaScript C#

Problem Description

Given an integer n, return the least number of perfect square numbers that sum to n.

A perfect square is an integer that is the square of an integer; in other words, it is the product of some integer with itself. For example, 1, 4, 9, and 16 are perfect squares while 3 and 11 are not.

Example 1:

Input: n = 12
Output: 3
Explanation: 12 = 4 + 4 + 4

Example 2:

Input: n = 13
Output: 2
Explanation: 13 = 4 + 9

Constraints:

  • 1 <= n <= 10⁴

Solution

Python Solution

class Solution:
    def numSquares(self, n: int) -> int:
        # Initialize dp array
        dp = [0] + [float('inf')] * n
        
        # For each number up to n
        for i in range(1, n + 1):
            # For each perfect square less than i
            j = 1
            while j * j <= i:
                dp[i] = min(dp[i], dp[i - j * j] + 1)
                j += 1
                
        return dp[n]

Time Complexity: O(n * sqrt(n))

For each number up to n, we check all perfect squares less than it.

Space Complexity: O(n)

We need an array of size n+1 to store intermediate results.

        // Initialize dp array
        int[] dp = new int[n + 1];
        Arrays.fill(dp, Integer.MAX_VALUE);
        dp[0] = 0;
        
        // For each number up to n

Approach Explanation

The solution uses dynamic programming to find the least number of perfect squares:

  1. Key Insight:
    • Each number can be formed by adding a perfect square to a smaller number
    • We can build solutions incrementally
    • Use dynamic programming to avoid recalculation
  2. Algorithm Steps:
    • Initialize dp array with infinity
    • For each number, try all possible perfect squares
    • Update with minimum count needed

Example walkthrough for n=12:

  • dp[0] = 0
  • dp[1] = dp[0] + 1 = 1 (1²)
  • dp[4] = 1 (2²)
  • dp[9] = 1 (3²)
  • dp[12] = min(dp[11]+1, dp[8]+1, dp[3]+1) = 3

Key insights:

  • Only need to check perfect squares <= i
  • Can reuse previously calculated results
  • Always exists a solution (Lagrange's theorem)
  • Answer is at most 4 (Lagrange's theorem)

Edge Cases:

  • Perfect squares (n = k²)
  • Small numbers
  • Large numbers
  • Numbers requiring maximum squares
