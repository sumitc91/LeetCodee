511. Game Play Analysis I

Jump to Solution: MySQL PostgreSQL MS SQL Server Oracle

Problem Description

Table: Activity 

+--------------+---------+
| Column Name  | Type    |
+--------------+---------+
| player_id    | int     |
| device_id    | int     |
| event_date   | date    |
| games_played | int     |
+--------------+---------+
(player_id, event_date) is the primary key of this table.
This table shows the activity of players of some games.
Each row is a record of a player who logged in and played a number of games (possibly 0) before logging out on someday using some device.

Write an SQL query to report the first login date for each player.

Return the result table in any order.

Example:

Input:

Activity table:
+-----------+-----------+------------+--------------+
| player_id | device_id | event_date | games_played |
+-----------+-----------+------------+--------------+
| 1         | 2         | 2016-03-01 | 5           |
| 1         | 2         | 2016-05-02 | 6           |
| 2         | 3         | 2017-06-25 | 1           |
| 3         | 1         | 2016-03-02 | 0           |
| 3         | 4         | 2018-07-03 | 5           |
+-----------+-----------+------------+--------------+

Output:

+-----------+-------------+
| player_id | first_login |
+-----------+-------------+
| 1         | 2016-03-01  |
| 2         | 2017-06-25  |
| 3         | 2016-03-02  |
+-----------+-------------+

Solution

MySQL Solution

SELECT 
    player_id, 
    MIN(event_date) AS first_login
FROM Activity
GROUP BY player_id;

Explanation:

  • Group records by player_id
  • Use MIN() to find the earliest event_date for each player
  • The result gives us each player's first login date

PostgreSQL Solution

SELECT 
    player_id, 
    MIN(event_date) AS first_login
FROM Activity
GROUP BY player_id;

Alternative Solution using Window Function:

SELECT DISTINCT
    player_id,
    FIRST_VALUE(event_date) OVER (
        PARTITION BY player_id 
        ORDER BY event_date
    ) AS first_login
FROM Activity;

MS SQL Server Solution

SELECT 
    player_id, 
    MIN(event_date) AS first_login
FROM Activity
GROUP BY player_id;

Alternative Solution using ROW_NUMBER:

WITH RankedDates AS (
    SELECT 
        player_id,
        event_date,
        ROW_NUMBER() OVER (
            PARTITION BY player_id 
            ORDER BY event_date
        ) AS rn
    FROM Activity
)
SELECT 
    player_id,
    event_date AS first_login
FROM RankedDates
WHERE rn = 1;

Oracle Solution

SELECT 
    player_id, 
    MIN(event_date) AS first_login
FROM Activity
GROUP BY player_id;

Alternative Solution using KEEP:

SELECT 
    player_id,
    MIN(event_date) KEEP (
        DENSE_RANK FIRST ORDER BY event_date
    ) AS first_login
FROM Activity
GROUP BY player_id;
